## 开篇定调

> **康宁是一家拥有世界级材料科学护城河的百年企业，但当前估值已完全脱离了基本面引力。110倍PE意味着市场正在以"AI光纤革命颠覆者"的叙事为一家年营收仅$156亿、净利润$16亿的成熟制造企业定价。巴菲特会说"等它跌到$40以下再来找我"；芒格会说"这是我今年见过的最荒唐的估值之一"。**

## 一、生意模式：百年材料科学的"配方+规模"生意

### 1.1 一句话概括

康宁是全球特种玻璃与陶瓷材料龙头，核心产品横跨显示面板玻璃基板（LCD/OLED）、光纤光缆、手机盖板玻璃（Gorilla Glass）、汽车尾气过滤陶瓷和生命科学耗材，客户覆盖消费电子、通信、汽车和生物医药四大终端市场。

### 1.2 产品/业务结构

| 业务板块 | 核心产品 | 市场地位 | FY2025营收占比 |
|:---|:---|:---|:---:|
| **光通信** | 光纤、光缆、连接器 | 全球光纤#1 | ~35% |
| **显示技术** | LCD玻璃基板、OLED基板 | 全球#1（与AGC/三星康宁） | ~30% |
| **特殊材料** | Gorilla Glass、先进光学 | 手机盖板垄断级 | ~15% |
| **环境技术** | 汽车陶瓷基板/过滤器 | 全球#1 | ~12% |
| **生命科学** | 实验室耗材、细胞培养 | 细分领先 | ~8% |

### 1.3 商业模式特征

- **收入模式**：B2B为主——向面板厂（京东方/TCL华星/三星）、光纤铺设商（AT&T/Verizon/中国移动）、手机厂（苹果/三星）、汽车厂供应核心材料。非消费品，无直接品牌溢价，完全靠技术性能和成本优势竞争。
- **核心特征——"配方经济"**：康宁真正的护城河在于**材料配方和精密成型工艺**。玻璃的化学成分、热历史、冷却曲线决定了最终性能。每款Gorilla Glass的配方是高度保密的，竞争对手逆向工程需要数年。
- **成本结构**：重资产。折旧摊销$13.5亿/年，占营收8.6%；CapEx$12.8亿/年。但毛利率稳定在31-40%区间，显示一定的成本传导能力。
- **增长方式**：有机创新驱动。康宁的每一代增长都来自材料发明的商业化：1879年灯泡玻璃→1947年电视显像管→1970年光纤→2007年Gorilla Glass→2020年代AI数据中心光纤。

### 1.4 巴菲特第一问

> "这个生意10年后还存在吗？它卖的产品10年后会是什么样？"

**玻璃和陶瓷是永恒的工业基础材料。** 10年后，世界仍需要光纤传输数据、玻璃基板制造显示器、特种玻璃保护手机。康宁的存续确定性很高（⭐⭐⭐⭐⭐），但产品形态和利润率的确定性只有中等（⭐⭐⭐）——面板技术可能被MicroLED颠覆，光纤需求高度周期化。

## 二、护城河分析

### 2.1 护城河五维度评估

| 护城河维度 | 评分 | 分析 |
|:---|:---:|:---|
| **品牌壁垒** | ⭐⭐ | B2B材料无消费者品牌溢价，但"Gorilla Glass"在手机营销中有半消费品属性 |
| **技术壁垒** | ⭐⭐⭐⭐⭐ | 材料配方+精密成型工艺，全球能稳定量产LCD玻璃基板的公司不超过5家 |
| **转换成本** | ⭐⭐⭐ | 面板厂更换玻璃供应商需重新调试产线，手机厂更换盖板玻璃需重新做跌落测试——有转换成本但并非不可逾越 |
| **规模优势** | ⭐⭐⭐⭐ | 高固定成本+高折旧→规模越大单位成本越低。小厂不可能在光纤和面板基板上与康宁拼成本 |
| **准入壁垒** | ⭐⭐⭐⭐ | 建一条LCD玻璃产线需$10亿+、3-5年调试期。特种玻璃的know-how是隐性知识，写在书本上也没用 |

### 2.2 护城河的核心支柱：材料配方×规模飞轮

康宁的护城河是一个**自我强化的飞轮**：

```
更深材料理解 → 更优配方/工艺 → 更高良率 → 更低成本 
    → 更多客户采用 → 更多出货量 → 更多规模效应 
    → 更多研发经费（$8亿/年，5.1%营收）→ 回到起点
```

这个飞轮的具体体现：
- **光通信**：康宁是全球唯一能在同一工厂内完成"预制棒→拉丝→成缆"全流程的企业，竞争对手只能做其中一段。预制棒技术（MCVD/OVD）的工艺know-how积累超过50年。
- **Gorilla Glass**：累计迭代到第7代（Victus 2），每代提升30-50%的抗跌落性能。苹果自iPhone 12起连续6代使用康宁玻璃，转换成本越来越高。
- **显示玻璃**：与三星显示合资成立三星康宁（Samsung Corning Precision Materials），用合资模式锁定最大客户。这是典型的"巴芒式"竞争壁垒——用利益绑定消灭竞争动机。

### 2.3 护城河在变宽还是变窄？

| 信号 | 方向 | 证据 |
|:---|:---:|:---|
| 毛利率趋势 | ↓ | 2016-2018年39-40% → 2023-2025年31-36%，面板价格战侵蚀 |
| 市场份额（光纤） | → | 全球#1维持，但中国长飞/亨通在逐步追赶 |
| 研发投入 | → | 稳定在$8-9亿/年（5-8%营收），无衰减迹象 |
| 客户粘性（Gorilla） | ↑ | 苹果连续6代使用，三星旗舰全系标配 |
| 竞争威胁（显示） | ↓ | 中国彩虹/东旭/旭虹在LCD基板取得突破，毛利率承压 |

**判断：护城河在光纤和特殊材料领域缓慢拓宽，在显示面板领域缓慢变窄。总体稳定但非单向利好。**

### 3.2 财务亮点与警报

#### 🟢 亮点一：重资产但高现金转化

康宁的OCF常年超过净利润：FY2025 OCF=$2.69B vs NI=$1.60B（OCF/NI=168%），因为折旧摊销$1.35B是非现金支出。真正的"所有者盈余"（OCF-维持性CapEx）约为$1.62B，是净利润的101%。

#### 🟢 亮点二：零净债务，资产负债表稳健

无长期有息负债（$0B LTD），现金$1.78B。虽然现金绝对值不高，但考虑到可随时发债的AAA级信用评级，财务风险极低。股权$11.8B，有形净资产扎实。

#### 🔴 警报一：净利润高度周期化且长期垫底

过去9年（FY2017-2025）中，有4年净利率低于6%，仅1年（FY2021）净利率超过13%。周期波谷时ROE跌至4-5%，这是典型的**大宗商品型周期股特征**——尽管康宁的产品是高科技材料。

#### 🔴 警报二：资本密集度过高，FCF/净利转化率波动剧烈

CapEx长期占OCF的50-70%。FY2018-2019年的CapEx投入几乎吞噬了全部OCF（FCF分别为$0.68B和$0.05B），FY2023的FCF也仅$0.61B。重资产属性意味着**股东几乎从未真正拿到过自由现金流**。

#### 🔴 警报三：营收增速已降至3%（FY2022-2025 CAGR）

$14.19B → $15.63B，三年CAGR仅3.3%。FY2025的19%营收增长是FY2023-2024面板寒冬后的V型反弹，不是趋势。**一个营收增速3%的成熟制造企业配110倍PE**，这在金融史上也是罕见的。

## 七、巴菲特×芒格双视角对话

### 7.1 芒格的视角

> 这公司让我想起了"漂亮50"时代（1972年）。当时市场认为"好公司无论什么价格都是好投资"，结果呢？宝丽来从$150跌到$1，麦当劳横盘了整整10年。康宁确实是好公司——配方垄断、规模飞轮、全球唯一的全产业链光纤制造商——但$1,766亿市值的康宁？对不起，就算你相信AI数据中心能让光纤需求翻三倍，我也只能给你估值$700亿，而且那还是最狂野的假设。剩下的$1,000亿是"叙事税"。110倍PE买一家净利率曾跌到3.9%的公司？这不是投资，这是宗教。

### 7.2 巴菲特的视角

> 芒格说得对，但我想补充一点：康宁的生意模式本身是巴菲特式的——"卖铲子"而非"淘金"。无论AI谁赢，数据都要在光纤上跑；无论手机谁第一，屏幕都需要玻璃保护。这是我喜欢的那种确定性。但问题出在价格上。2000年可口可乐PE 60倍时我选择持有而非买入，结果等了12年才回本。今天康宁110倍PE，比我当年犯的错还夸张。如果康宁跌到$35-40（PE~20x），我会立即打电话给我的经纪人。在那之前，我只愿意远远看着。
