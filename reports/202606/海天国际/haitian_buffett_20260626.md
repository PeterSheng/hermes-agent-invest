**巴菲特会说：**
>"9年里有8年OCF接近或超过净利润，9年累计OCF/NI约102%。这意味着——海天赚的每一块钱利润，都变成了一块钱现金进了银行账户。没有应收账款陷阱，没有存货减值的把戏。唯一的例外是2024年（OCF/NI=68%），是因为收入大增23%导致营运资本占用——这在成长期公司中完全正常，不是红旗。"

### ★★ 巴菲特式ROE/ROIC分析：海天被低估的核心证据

```

巴菲特衡量ROE要看"来源"：
  ├── 高杠杆推高的ROE？ → 不是。海天负债率仅35%，几乎无有息负债

但问题是：11.6%的ROIC意味着"护城河够宽但不是不可逾越的"。

```

## 四、护城河分析：海天 vs 福耀，谁的护城河更深？

### 4.1 福耀的护城河

```
福耀 = "汽车玻璃=福耀" 的品牌垄断
├── 认证壁垒：每个新车型玻璃需2-3年认证周期
├── 地理壁垒：玻璃重+易碎→工厂必须靠近整车厂<300km
├── 规模壁垒：全球最大单一工厂（福清）→成本无人能及
├── 客户粘性：一旦定点，跟车型5-7年→换供应商成本极高
└── ★ 护城河深度：9/10

福耀护城河的核心逻辑：
"整车厂不敢换供应商——换玻璃供应商意味着重新做碰撞测试、
重新认证、重新调校产线。代价远超省下的几个点成本。"
这是一种"高转换成本+认证周期"的双重锁定。
```

### 4.2 海天的护城河

```
海天 = "全球#1注塑机"的规模优势
├── 规模壁垒：年产36,000台→采购钢铁/液压件价格比对手低5-10%
├── 渠道壁垒：130+国家→服务网络密度远超任何对手
├── 品牌认知："海天"在中国=注塑机的代名词
├── 成本优势：宁波制造基地→劳动力+供应链集群效应
├── 技术积累：二板机、全电动机、6600吨级超大型机
└── ★ 护城河深度：6.5/10

但海天护城河的核心弱点：
"客户从海天切换到伊之密/震雄，成本并不高。
一个塑料厂老板可以这周买海天、下周买伊之密——
没有认证周期，没有法规壁垒。切换成本≈零。"
```
