## 一、生意模式：企业服务的"路线经济学"

### 1.1 一句话概括

Cintas是北美最大的企业制服和工作场所服务商——每天有11,000+辆卡车按固定路线拜访100万+企业客户，收取脏制服/地垫/毛巾→运回洗衣厂清洗→下次拜访时送回干净的。这个模式的核心是**路线密度经济**：同一条路线上客户越多，每站成本越低，利润率越高。

### 1.3 路线经济学的数学

```
假设一条路线服务100个客户：
  每个客户年均消费 $8,000
  路线上卡车+司机年成本 $120,000
  洗衣厂处理成本 $380,000
  
  路线收入: 100 × $8,000 = $800,000
  路线成本: $120,000 + $380,000 = $500,000
  路线毛利: $300,000 (37.5%)
  
如果在同一条路线上增加10个客户（无需新增卡车/司机）：
  路线收入: 110 × $8,000 = $880,000
  路线成本: $120,000 + $418,000 = $538,000
  路线毛利: $342,000 (38.9%)
  
→ 每增1个同路线客户 = 几乎纯利润！
```

> 这正是Cintas护城河的精髓：在同一地理区域积累的客户越多，单位经济越好——新进入者在任何区域都无法匹敌Cintas的密度优势。

### 1.4 巴菲特第一问

> "这个生意10年后还存在吗？"

**确定性：★★★★★**。企业永远需要员工穿制服、门口放地垫、洗手间有毛巾。只要商业活动存在，Cintas的路线服务就有需求。这是一个与GDP同步增长、几乎无周期性的永续生意。

### 核心支柱

#### 🚛 路线密度——无法复制的终极壁垒

- 11,000+条已盈利路线覆盖北美→新进入者需要同时开数千条亏损路线来竞争
- 每增加一个同路线客户 = 纯利润（无需新增固定成本）
- 1M+客户基础 → 任何一个行业/区域的饱和都不可能被竞品复制

#### 🏭 洗衣厂网络——资本壁垒

- 400+工业洗衣厂 → 总投资$50亿+
- 每个洗衣厂服务周边50-100条路线
- 新进入者建厂 → 至少$2-5亿启动资金 + 3-5年达到盈亏平衡
