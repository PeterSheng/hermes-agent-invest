# 兴齐眼药(300573)巴菲特视角深度分析

> **分析日期:** 2026-06-07 | **股价:** 36.95元 | **市值:** ~132亿 | **PE(TTM):** 17.46x | **PB:** 6.81x

---

## 一、开篇定调

**"兴齐眼药是一个罕见的组合：81%毛利率的眼科药垄断品种+17倍PE。如果巴菲特的搭档芒格还活着，他可能会说：'这是一家你只需要搞懂一个产品就能理解整个公司的企业——硫酸阿托品滴眼液，中国3亿近视儿童家长的刚需。'"**

兴齐眼药专注眼科药物30年，拥有60个眼科药批文。2024年3月获批的中国首个延缓儿童近视进展的硫酸阿托品滴眼液，让公司从一家"小而稳"的眼科药企蜕变为"有独家品种驱动的成长股"。2025年净利暴增106%至6.96亿、ROE跳升至39%、毛利率升至81%。而市场给它的定价只有17倍PE——这是我在44家公司中见到的"最便宜的超高ROE公司"。

## 二、生意模式

### 2.1 产品矩阵

| 品类 | 代表产品 | 地位 |
|:---|:---|:---|
| **延缓近视** | 0.01%硫酸阿托品滴眼液 | 🥇 国内唯一获批，2024年3月上市 |
| **干眼治疗** | 环孢素滴眼液(II) | 🥇 中国首个干眼症环孢素制剂 |
| **抗感染** | 左氧氟沙星、妥布霉素滴眼液 | 成熟品种 |
| **抗炎** | 氟米龙、普拉洛芬滴眼液 | 成熟品种 |
| **抗青光眼** | 毛果芸香碱等 | 成熟品种 |
| **其他** | 散瞳药、人工泪液等 | 覆盖十个细分类别 |

### 2.2 生意的核心引擎：阿托品滴眼液

```
中国近视儿童：约3亿（6-18岁）
适用低浓度阿托品人群：约1-2亿
年治疗费：约2000-3000元/人
潜在市场规模：2000-6000亿
当前渗透率：<1%

兴齐眼药：目前唯一获批企业
```

**这是一个"合法垄断窗口期"的生意**——在竞品获批之前，兴齐独享一个千亿级潜在市场。

### 2.3 生意模式评分：★★★★★

眼科药天生具有巴菲特喜欢的所有特质：患者复购率高（慢性病需持续用药）、品牌认知度强、毛利率惊人、无需大量资本开支。

## 三、护城河分析

| 维度 | 评分 | 说明 |
|:---|:---:|:---|
| **品牌壁垒** | ⭐⭐⭐⭐ | 医生处方习惯+眼科专业品牌30年积累 |
| **转换成本** | ⭐⭐⭐⭐ | 阿托品需要持续使用2-3年，中途换品牌有风险 |
| **技术壁垒** | ⭐⭐⭐⭐ | 制剂工艺（无防腐剂单剂量包装）+临床数据护城河 |
| **规模优势** | ⭐⭐⭐ | 25亿营收在医药行业不算大 |
| **准入壁垒** | ⭐⭐⭐⭐⭐ | **核心壁垒。** 眼药审批周期长（3-5年），竞品即使研发完成也需数年才能获批 |

**综合护城河：⭐⭐⭐⭐（强。先发优势+审批壁垒构筑窗口期垄断）**

### 护城河可持续性判断

阿托品分子本身无专利保护，护城河的核心在于：
1. **审批时间差：** 竞品（恒瑞、欧康维视等）即使完成临床，获批仍需2-4年
2. **临床数据积累：** 兴齐拥有最长随访期的中国儿童近视延缓真实世界数据
3. **渠道铺设先发：** 已覆盖全国医院眼科和视光中心

**判断：护城河在3-5年内牢固，之后随着竞品获批会收窄。但品牌+渠道+数据积累仍能维持一定优势。**

### 结论

兴齐眼药是44家公司中**风险收益比前三的标的**。核心逻辑简单清晰：在中国3亿近视儿童家长的需求驱动下，阿托品滴眼液独占3-4年窗口期。17倍PE定价了一个"正常增长"的眼科药企，但忽略了"独家品种爆发"的期权价值。

**最大风险：竞品提前获批。** 一旦有其他阿托品获批，市场可能从"垄断定价"变为"多家竞价"。如果竞品2028年才获批，兴齐还有2-3年高增长窗口。如果竞品2027年就获批（通过优先审评），安全边际会缩窄。
