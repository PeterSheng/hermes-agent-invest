## 一、生意模式：中国2亿过敏患者的"唯一口服解药"

### 1.1 一句话概括

我武生物做的是**脱敏治疗**——让尘螨过敏患者每天在舌下滴几滴药，经过2-3年治疗后，身体不再对尘螨过敏。**它不是"治症状"的抗过敏药（氯雷他定），而是"改体质"的免疫疗法——是整个过敏治疗金字塔的顶端。**

### 1.2 产品壁垒：三道铜墙铁壁

| 壁垒 | 说明 |
|:---|:---|
| **药证壁垒** | 中国只有3张尘螨脱敏药证：我武(舌下滴剂)、ALK(丹麦，注射)、Allergopharma(德国，注射) |
| **剂型优势** | 舌下滴剂 vs 注射剂 = 在家用药 vs 去医院 = 无痛 vs 疼痛 = 儿童可用 vs 儿童恐惧 |
| **治疗锁定** | 脱敏治疗周期2-3年，一旦开始不会轻易中断——等于"签约3年的订阅用户" |

### 1.3 市场规模

中国尘螨过敏患者约**2亿人**，其中适合脱敏治疗的中重度患者约5,000万人。目前接受脱敏治疗的患者仅**几十万人**——渗透率不到1%。对比欧洲脱敏治疗渗透率>10%，市场空间是**10倍以上**。

### 1.4 成本结构——"印钞机"的经典配方

| | 2025年 | 占营收 |
|:---|:---|:---|
| 营收 | 10.62亿 | 100% |
| **营业成本** | **0.51亿** | **4.8%** |
| 毛利润 | 10.11亿 | 95.2% |
| 销售费用 | 4.03亿 | 37.9% |
| 研发费用 | 1.22亿 | 11.5% |
| 营业利润 | 4.41亿 | 41.5% |
| 净利润 | 3.72亿 | 35.0% |

**每100元营收，生产成本不到5元——95%毛利率是中国上市公司中最高的一档。** 销售费用38元（学术推广+医生教育），研发11元，净利润35元。这种成本结构和新产业（化学发光）、正海生物（再生医学）同属"知识产权变现"的顶级商业模式。

### 2.1 护城河五维度

| 维度 | 评分 | 分析 |

| **产品优势** | ⭐⭐⭐⭐⭐ | 唯一舌下含服——安全、便捷、儿童友好 |
| **转换成本** | ⭐⭐⭐⭐⭐ | 患者一旦开始2-3年疗程，中途不会换 |
| **品牌/医生心智** | ⭐⭐⭐⭐ | 粉尘螨滴剂=脱敏治疗的代名词 |
| **规模优势** | ⭐⭐⭐ | 10亿营收还小，但在细分赛道已经是绝对龙头 |

### 2.2 护城河的真实深度

> **如果有人今天想复制我武生物，他需要：**
> ① 花5-7年完成临床试验和药品审批
> ② 花数亿元建立GMP生产线
> ③ 花数年搭建覆盖全国的学术推广团队
> ④ 说服已经用了我武产品多年的医生改用新品牌
>
> **在以上四点全部完成之前，我武已经又领先了5-7年。这不是护城河——这是"时间机器"。**
