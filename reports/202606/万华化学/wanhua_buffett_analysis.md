# 万华化学(600309)巴菲特视角深度分析

> **分析日期:** 2026-06-07 | **股价:** 72.46元 | **市值:** ~2,268亿 | **PE(TTM):** 17.23x | **PB:** 2.11x

---

## 一、开篇定调

**"万华化学是中国化工行业皇冠上的明珠——全球MDI老大、技术自主、管理卓越、全球布局。但明珠也有蒙尘的时候：毛利率从40%跌到14%、ROE从51%跌到12%、七年营收翻倍利润却原地踏步。这就是化工周期的宿命——再好的企业也无法逃脱产品价格的引力。"**

万华化学是中国极少数真正具备全球竞争力的化工企业。MDI（二苯基甲烷二异氰酸酯）产能全球第一，技术自主可控（第八代MDI技术），产品覆盖聚氨酯、石化、精细化工、新材料、电池材料五大板块。2025年营收2032亿，是中国最大的化工上市公司。但MDI价格从2021年高点腰斩，导致利润连续三年下滑。

## 二、生意模式

### 2.1 万华做什么？

| 业务板块 | 核心产品 | 2025年收入占比 |
|:---|:---|:---:|
| **聚氨酯** | MDI、TDI、聚醚多元醇 | ~45% |
| **石化** | C2/C3/C4烯烃衍生物、PVC | ~30% |
| **精细化学品与新材料** | 特种胺、ADI、TPU、PC、PMMA、POE | ~18% |
| **电池材料** | 磷酸铁锂、钠电材料、硅碳负极 | ~5% |
| **其他** | 功能化学品、表面材料等 | ~2% |

### 2.2 MDI的生意本质

MDI是聚氨酯的核心原料，下游应用于冰箱保温、建筑保温、汽车、胶粘剂、涂料等。全球MDI市场约800万吨/年，万华占约30%份额。

MDI的特殊之处在于：
- **技术门槛极高：** 全球只有万华、巴斯夫、科思创、亨斯迈、陶氏五家企业能大规模生产
- **光气化工艺危险复杂：** 新建MDI装置审批极其困难
- **装置投资巨大：** 单套40万吨MDI装置投资约100亿+

这形成了天然的寡头垄断格局。

### 2.3 生意模式评分：★★★★☆

MDI寡头格局+技术自主+全球布局使其成为化工界的"茅台"。但产品价格随宏观周期波动，且近三年进入下行通道。

### 4.2 核心矛盾：七年营收翻倍，利润原地踏步

```
2017年：营收531亿 → 净利111亿 → 净利率21%
2025年：营收2032亿 → 净利125亿 → 净利率6.9%

营收增长283%，利润仅增长13%。
每1元收入创造的利润从0.21元降到0.06元。
```

### 4.3 五大财务信号

| 信号 | 状态 | 解读 |
|:---|:---:|:---|
| **毛利率** | 🔴 连降7年 | 40%→14%，MDI价格下行周期是主因 |
| **净利率** | 🔴 持续压缩 | 21%→7%，石化业务亏损拖累 |
| **ROE** | 🔴 腰斩 | 51%→12%，从超级回报回归平庸 |
| **OCF** | 🟢 依然强劲 | 331亿远超125亿净利 |
| **CAPEX** | 🟡 巨大 | 年300亿+，FCF接近零或为负 |

### 4.4 盈利质量好但资本开支吞噬一切

2024-2025年OCF约300-330亿/年，但CAPEX也是300亿+/年。FCF≈0。赚的钱全部变成了新的装置和产能。

## 五、行业与竞争

全球MDI三强格局：万华(30%) > 巴斯夫(20%) > 科思创(18%)。供给端集中，需求端分散。但2021年后中国多套新装置投产导致供给过剩。

### 结论

万华化学是中国化工行业最值得尊敬的公司——全球MDI老大、技术自主、管理层卓越。但化工周期的铁拳同样不会放过它。当前18倍PE对应ROE 12%——不便宜。**如果MDI价格回升+新材料兑现，股价有翻倍空间。如果继续过剩，18倍PE还有压缩风险。这是一个"在周期底部等待黎明"的标的，属于"可以欣赏但需要耐心等待"的类别。**
