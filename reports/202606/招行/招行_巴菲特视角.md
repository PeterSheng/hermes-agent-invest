## 九、巴菲特式终极追问

### 9.1 招行="中国富国"吗？

| 维度 | 富国(巅峰) | 招行(2025) | 
|------|-----------|-----------|
| 低成本存款 | ~45%活期 | **~58%** ✅ |
| 交叉销售 | 户均8产品 | 户均~5 ⚠️ |
| 零售聚焦 | 社区银行之王 | 零售AUM之王 ✅ |
| ROE | 13-17% | **13.4%** ✅ |
| 保守文化 | 极端保守 | 相对稳健 ⚠️ |
| 丑闻 | 虚假账户2016 | 田惠宇2022 ⚠️ |

> 🔑 最接近富国的中国银行，但非复制品。招行护城河更依赖"牌照+负债成本"，富国更依赖"交叉销售+社区网络"。

### 9.2 PB<1 = 毁灭价值？市场错了吗？

```
COE(权益成本) = 2.5% + 0.8×6% = 7.3%
ROE = 13.4% > 7.3% × 1.8倍
→ 理论PB应 > 1.5x，实际 0.80x
```

**市场定价逻辑链：**
```
NIM↓ → ROE跌破10% → 逼近COE → PB<1合理
```

**巴菲特的三条反驳：**

1. **利率不会无限下行**：中国不会日本化——人口结构不同、政策空间更大
2. **低利率是护城河放大镜**：NIM=1.5%时25bp成本优势占ROE的2pp，不是橡皮擦
3. **PB<1是情绪定价**：40%"中国折价"反映宏观焦虑，非招行基本面

### 9.3 模拟决策：巴菲特会买招行吗？

✅ **会：** 能力圈内 | 持久护城河 | PB 0.80x安全边际 | 5%+股息率 | 长期复利

⚠️ **不会：** 宏观依赖(GDP↓) | 政策工具化(让利实体) | 治理风险(田惠宇) | 富国教训(丑闻后减持)

> 🎯 **如果巴菲特投资中国，招行一定在20个打孔位候选名单上。** 但最终取决于他对中国宏观判断。

### 9.4 巴菲特合理估值

```
正常化PE ≈ 8.5 + 2×g = 8.5 + 2×3 = 14.5
内在价值 ≈ 5.70 × 14.5 = 82.65元

考虑：银行杠杆折价30% → 57.86元
再考虑：中国折价30% → 40.50元

巴菲特合理买入区：PB 1.0-1.2x → 43-52元
当前36元：安全边际20-44%
```
