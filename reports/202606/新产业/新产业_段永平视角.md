## 一、生意本质：化学发光是什么？

### 1.1 一句话讲清楚

> **化学发光 = 医院的"血液/体液自动检测流水线"。**

抽一管血 → 放进新产业的仪器 → 15-30分钟自动出几十项指标（甲状腺、肿瘤标志物、性激素、心肌标志物、传染病……）。

**不是药，是诊断工具。** 药是消费品，诊断是基础设施。

### 1.2 商业模式的本质：剃刀+刀片（Razor & Blade）

```
┌─────────────────────────────────────────────────────────┐
│              新产业商业模式拆解                            │
├─────────────────┬───────────────────────────────────────┤
│  仪器（剃刀）     │  试剂（刀片）                           │
├─────────────────┼───────────────────────────────────────┤
│  一次性销售        │  持续消耗，每个测试用一份                 │
│  低毛利/亏本卖     │  高毛利（70-80%），真正的利润来源         │
│  装机是"获客成本"  │  试剂消耗 = 年费收入                      │
│  一台仪器用5-8年   │  单台仪器年试剂产出：30-80万元            │
├─────────────────┼───────────────────────────────────────┤
│  收入占比 ~25%    │  收入占比 ~75%，利润占比 ~90%              │
│  全球累计装机 ~4万台│  2025试剂收入 ~34亿                     │
└─────────────────┴───────────────────────────────────────┘
```

### 1.3 和苹果 iOS 生态的对比

| 维度 | 苹果 iPhone + App Store | 新产业 仪器+试剂 |
|------|------------------------|-------------------|
| 硬件 | iPhone（利润不错） | 化学发光仪（微利/亏损） |
| 生态锁定 | iOS + App Store（30%抽成） | 封闭试剂系统（只能用自产试剂） |
| 转换成本 | 极高（数据、习惯、配件） | 中高（重验证+重培训+IT对接） |
| 边际成本 | App分发成本趋近于零 | 试剂生产有物料成本（~20%） |
| 定价权 | ★★★★★ 强 | ★★★☆☆ 有但被集采削弱 |
| 竞争对手 | 三星/华为 | 罗氏/雅培/贝克曼/西门子 |
| 政策风险 | 反垄断（欧盟DMA） | 集采（更狠，直接砍价50-70%） |

> **关键差异：苹果的30%抽成是平台税，政府最多要求开放侧载，不会直接定价。但集采是政府直接规定试剂价格——等于政府替你定价，对"剃刀刀片"模式是降维打击。**

### 1.4 为什么化学发光比生化诊断值钱？

| | 生化诊断 | 化学发光（CLIA） |
|---|---|---|
| 技术壁垒 | 低，国产替代已完成 | 中高，进口仍主导 |
| 仪器单价 | 10-50万 | 30-120万 |
| 试剂毛利率 | 50-60% | 70-85% |
| 国产化率 | >70% | ~30-40% |
| 集采冲击 | 已落地，降价70-80% | 正在落地，降价50-70% |
| 代表公司 | 迈瑞、迪瑞 | 新产业、安图、迈瑞 |

**化学发光是IVD皇冠上的明珠——技术更难、毛利更高、替代空间更大。这也意味着：集采的刀，第一个砍的就是这块肥肉。**

## 二、差异化：新产业凭什么打"四大家"？

### 2.1 全球竞争格局

| 玩家 | 全球份额 | 中国份额 | 核心优势 |
|------|---------|---------|---------|
| 罗氏 Roche | ~30% | ~25% | 菜单最全（200+项目），品牌最强 |
| 雅培 Abbott | ~20% | ~15% | 传染病检测王者 |
| 贝克曼 Beckman Coulter | ~12% | ~10% | 高通量仪器，内分泌强 |
| 西门子 Siemens | ~10% | ~8% | 仪器稳定，德国品质 |
| **新产业 Snibe** | **~3-4%** | **~8-10%** | **国产第一，速度+成本+本土服务** |
| 安图 Autobio | ~2% | ~6% | 国产第二，微生物协同 |
| 迈瑞 Mindray | ~2% | ~5% | 全线IVD+监护仪渠道协同 |

### 2.2 新产业的差异化清单

```
★ 速度：全球最快单机测速（MAGLUMI X8: 600测试/小时）
   → 罗氏cobas e801: 300测试/小时，雅培Alinity: 200测试/小时
   → 对大型医院检验科 = 实打实的效率提升

★ 价格：试剂价格比进口低40-60%
   → 集采前，同样的肿瘤标志物检测，新产业成本是罗氏的1/3

★ 服务：本土24小时响应 vs 进口48-72小时
   → 仪器坏了对医院是灾难，本土服务是硬壁垒

★ 试剂菜单：~170项注册证，国产最多
   → 覆盖甲功、肿瘤、性激素、心肌、传染病、代谢六大类
   → 但vs罗氏200+项仍有差距

★ 海外布局：150+国家装机，海外收入占比~35%
   → 不受国内集采影响，在印度/俄罗斯/东南亚是主力玩家
```

### 2.3 差异化壁垒深度评估

段永平最喜欢问：**"竞争对手能不能复制？要花多少钱？要花多久？"**

| 差异化 | 可复制性 | 复制时间 | 段氏评分 |
|--------|---------|---------|:---------:|
| 仪器速度 | ★★★ 可复制 | 3-5年 | 临时优势，不是护城河 |
| 低价 | ★ 最容易复制 | 1-2年 | 不是护城河！迈瑞有钱烧 |
| 本土服务 | ★★ 中等 | 2-3年 | 迈瑞渠道更强 |
| 试剂菜单 | ★★★★ 较难 | 5-8年（注册证周期） | 有壁垒但非永久 |
| 海外渠道 | ★★★ 较难 | 3-5年 | 先发优势 |
| 装机存量 | ★★★★★ 最难 | 需要医院愿意换 | **这才是真壁垒！** |

> **核心结论：新产业最大的护城河不是技术，不是价格，不是速度——是已经装进医院的4万多台仪器。每一台=一个收费站，每年产出30-80万试剂收入。竞争对手想抢，先让医院把现有仪器扔了——绝大多数医院不会主动这么做。**

## 三、定价权：集采来了，还剩多少？

### 3.1 IVD集采时间线

```
2021年：安徽先行先试，化学发光试剂集采，平均降幅47%
2022年：江西牵头23省肝功生化集采，平均降幅68%
2023年：安徽牵头25省化学发光（传染病+性激素）集采，平均降幅53%
2024年：江西牵头24省肾功+心肌集采，平均降幅70%+
2025年：肿瘤标志物等更多品类纳入，集采全面铺开
```

### 3.2 新产业净利率——定价权消失的铁证

| 年份 | 净利率 | 同比变化 | 发生了什么？ |
|------|--------|---------|-------------|
| 2021 | 38.0% | — | 集采尚未全面落地 |
| 2022 | **43.6%** | **+5.6pp** | 规模效应+高毛利试剂放量 |
| 2023 | 42.0% | -1.6pp | 安徽集采开始执行 |
| 2024 | 40.3% | -1.7pp | 多省集采铺开 |
| 2025 | **35.4%** | **-4.9pp** | 集采全面落地 + 价格战加剧 |

**净利率从43.6%掉到35.4%，两年蒸发8.2个百分点——将近1/5的利润消失。**

### 3.3 定价权终极测试：量化模型推演

```
模型设定：
  集采前：试剂毛利率80%，仪器毛利率20%，综合毛利率65%
  试剂占收入75%，仪器25%，固定费用（销售+管理+研发）约25%

┌────────────────┬──────────┬──────────┬──────────┬──────────┐
│  集采降价幅度    │   -30%   │   -50%   │   -70%   │   -80%   │
├────────────────┼──────────┼──────────┼──────────┼──────────┤
│ 新试剂毛利率     │   71%    │   60%    │   33%    │   0%     │
│ 综合毛利率       │   59%    │   50%    │   30%    │    5%    │
│ 净利率（量不变）  │   34%    │   25%    │    5%    │  亏损    │
│ 净利率（量+50%） │   34%    │   25%    │    5%    │  亏损    │
│ 净利变化(量不变) │  -16%    │  -37%    │  -87%    │  亏光    │
│ 净利变化(量+50%)│  +26%    │   -6%    │  -81%    │  亏光    │
└────────────────┴──────────┴──────────┴──────────┴──────────┘
```

**关键发现：**
- ✅ 降价30%：扛得住，量增可补价降（净利甚至可能增长）
- ⚠️ **降价50%：量增50%才能维持利润——这正是2025年发生的事**
- ❌ 降价70%：商业模式崩溃，量翻倍都救不回来
- ☠️ 降价80%：试剂零毛利，公司变公益机构

> **2025年净利-11.4%的本质：集采降价幅度（~50%）跑赢了量的增长（~10-15%）。"以量补价"的逻辑失效了。**

### 3.4 与段永平经历过的价格战对比

段永平做小霸王和步步高时，经历过中国家电的惨烈价格战。他的核心洞察：

> **"价格战打到最后，只有成本最低的那家能活。"**

- 新产业的成本比罗氏/雅培低吗？**是的——人工、制造、销售费用全方位更低。**
- 但集采是政府主导，不是市场竞争——目标不是让成本最低的活，而是统一降到"合理价格"
- **关键是：那个"合理价格"是否还允许合理利润？**

从安徽/江西实践看：集采中标价约为出厂价的50-60%，仍远高于制造成本（COGS约20%），**所以不至于亏本，但暴利时代结束。**

## 四、用户黏性：装了仪器就锁住了吗？

### 4.1 医院转换成本的真实门槛

**表面逻辑：** 仪器封闭系统，试剂不通用 → 锁定。

**实际情况：** 转换成本没有想象的高。

```
医院换供应商的真实门槛：

① 仪器重置成本：30-120万/台
   三甲医院：≈一台CT/MR的零头，不是大事
   县级医院：是一笔钱，但可承受

② 试剂重新验证：每个检测项目需做精密度/准确度/线性范围验证
   170个项目全部验证：1-2个月
   → 这是真门槛！检验科主任最怕麻烦

③ LIS/HIS系统对接：仪器要和医院信息管理系统联通
   IT改造：2-4周，有麻烦但可解决

④ 人员培训：1-2周，不算大问题

⑤ 结果一致性：新旧仪器参考范围不同，临床医生需重新适应
   → 隐形成本
```

### 4.2 对比福耀玻璃的转换成本

| 维度 | 福耀汽车玻璃 | 新产业化学发光 |
|------|------------|---------------|
| 客户关系 | OEM前装，认证周期3-5年 | 医院招标/集采，周期1-2年 |
| 转换成本 | **极高**——重新碰撞测试+整条产线调整 | **中高**——重验证+重培训+IT对接 |
| 议价能力 | 福耀全球30%份额，主机厂怕断供 | 国产三强+进口四大家，医院有选择 |
| 定价权 | ★★★★ 强 | ★★★ 中等偏弱 |
| 周期性 | 跟汽车周期 | 跟医保政策周期 |

> **新产业的转换成本比福耀低至少一个量级。福耀的护城河是"全球只有我"，新产业的护城河是"换起来有点麻烦"——差距巨大。**

### 4.3 "装机量=护城河"的谬误

很多人说：新产业装了4万多台仪器=护城河。**错。**

装机量只有当"医院不愿意换"时才是护城河。集采正在改变这一点：

- **集采前：** 医院自己谈价，换供应商麻烦 → 不换
- **集采后：** 医保局统一定价，医院只能用中标品牌 → 新产业没中标？换！医保局帮你换

> **集采把"医院不愿意换"变成了"医保局要求你换"——装机量护城河被政策炸开了一道口子。**

## 五、飞轮效应：被集采打断了吗？

### 5.1 化学发光的理想飞轮

```
 装仪器 ──→ 试剂消耗 ──→ 利润累积
    ↑                      ↓
    │                      ↓
 更多医院认可 ←── 研发更多试剂菜单 ←── 再投入研发
    │                      ↓
    │                  仪器迭代升级
    │                      ↓
 装机量扩大 ←──── 新产品竞争力增强
```

**2019-2023年飞轮运转完美：** 装机2万→4万台，试剂120→170项，试剂收入15→34亿，ROE 18.9%→23.7%。

### 5.2 集采打断了什么？

```
 装仪器 ──→ 试剂消耗 ──→ 利润累积
    ↑           ⚡            ↓
    │       【试剂降价50%】     ↓
    │                      【利润缩水】
    │                          ↓
 更多医院认可 ←── 研发更多试剂 ←─?─ 研发投入？
    ?                          ↓
【集采中标=被迫降价换量】    仪器迭代？
```

**打断的是最核心环节："试剂消耗 → 利润累积"。**

- 过去一台仪器：年产出50万试剂收入，毛利40万（80%），净利18万（36%）
- 现在一台仪器：试剂价砍半→年产出25万，毛利15万（60%），净利5万（20%）

> **单台仪器的经济价值直接腰斩。飞轮还在转，但转速减半。**

## 六、2025年净利下滑：一次性还是结构性？

### 6.1 利润下滑拆解

```
2024年净利：18.3亿 → 2025年净利：16.2亿 → 减少2.1亿（-11.4%）

├─ 集采试剂降价影响：    -3.5亿（试剂收入~34亿，均价降~10%）
├─ 以量补价（销量+15%）：  +1.5亿
├─ 海外增长（+20%）：      +1.0亿（不受集采影响）
├─ 仪器收入：             约持平
├─ 费用压缩（降本增效）：  +0.5亿
└─ 其他（减值/汇率/税）：  -0.6亿
─────────────────────────────────────────
合计：                     -2.1亿 ✓
```

### 6.2 综合判断矩阵

| 论据 | 指向 |
|------|:---:|
| 2025年是多个省份集采落地首年，价格冲击集中释放 | ✅ 一次性 |
| 2026年无新大规模集采增量（主要品类已纳入） | ✅ 冲击见顶 |
| 试剂毛利率从80%→60-70%是永久性的，不会反弹 | ❌ 结构性 |
| 海外收入占比35%且在增长，部分对冲国内集采 | ✅ 缓冲垫 |
| 集采范围只会扩大不会缩小，未来纳入更多项目 | ❌ 持续压力 |
| 国产替代仍有空间（从30%→50%），量增可期 | ✅ 增量逻辑 |
| 小玩家出清，龙头份额集中 | ✅ 长期利好龙头 |
| 试剂COGS仅~20%，降价50%毛利率仍>50% | ✅ 不会亏本 |

**结论：七分结构性，三分一次性。**

毛利率的下降是永久性的——政府定价一旦确立不会回到市场化定价。但2025年-11.4%的降幅包含了集采首年的一次性冲击（渠道清库存、过渡混乱），2026年降幅有望收窄甚至企稳。

**关键窗口：2026年中报。净利继续下滑>5%=结构性恶化持续。企稳/微增=一次性冲击消化完毕。**

## 七、段永平式终极追问

### 7.1 "10年后这家公司还在吗？"

**化学发光需求10年后还在吗？**
✅ **一定在。** 只要看病就需要抽血检测。全球IVD年增速5-7%，化学发光增速8-12%，确定性最高的医疗赛道之一。

**新产业10年后还在吗？**
⚠️ **大概率在，但不一定是现在的地位。**
- 集采深化：行业变微利，剩3-5家存活，新产业是存活者之一
- 国产替代加速：市占率从10%→20%
- 技术颠覆风险：质谱、POCT可能吃掉部分市场
- **最大威胁：迈瑞**——有渠道、有钱、有品牌、有全线IVD矩阵，是最可怕的竞争者

### 7.2 "化学发光会不会被新技术颠覆？"

| 替代技术 | 威胁 | 时间窗口 | 理由 |
|---------|:---:|---------|------|
| 质谱（LC-MS/MS） | ★★☆ | 10年+ | 太贵太慢太复杂，只适合特殊项目 |
| NGS（高通量测序） | ★☆☆ | 不构成威胁 | 做基因的，和蛋白检测互补不替代 |
| 微流控POCT | ★★★ | 5-10年 | 可能吃掉急诊/基层部分市场 |
| 电化学发光（罗氏ECL） | ★★★★ | 现在 | 罗氏独有，灵敏度高但成本高，反给新产业机会 |
| AI+多组学 | ★★☆ | 10-15年 | 太远期，且AI需要发光仪生成数据 |

> **化学发光5-10年内不会被颠覆。15年+任何事情都可能发生，但现在不用太担心。**

### 7.3 "如果试剂价格砍到1/3，能赚钱吗？"

```
试剂价格降至30%：
  新试剂毛利率 = 33%（还能赚但很薄）
  综合毛利率 = 30%
  扣除固定费用25% → 净利率 ≈ 5%
  以2025年45.8亿营收算 → 净利 ≈ 2.3亿
  当前市值333亿 → PE ≈ 145倍 → 灾难
```

**但这种情况发生概率极低：**
1. 医保局目的不是"杀行业"而是"挤水分"——全砍死谁供应？
2. 化学发光试剂有硬成本地板：物料成本~20-25%，砍到30%以下就亏本
3. 以量换价必须兑现"量"——否则企业退出，集采失败

**更可能的结果：试剂毛利率稳定在55-65%，净利率25-30%，ROE 15-18%。不是一个暴利生意了，但仍然是一个好生意。**

### 8.3 最终判断：段永平大概率不会买

**五个核心理由：**

> **① "不懂不做"——不是不懂生意，是不懂政策。** 集采什么时候集、集什么、砍多少——完全取决于医保局，企业没有话语权，投资者更预测不了。段永平："如果生意靠政府政策活着或被政策杀死，我不想碰。"

> **② 定价权缺失。** 段永平最爱有定价权的公司——苹果定iPhone价，茅台定飞天价。新产业的价是医保局定的。没有定价权的公司，"赚的是辛苦钱"。

> **③ 护城河不够深。** 和茅台（品牌）、苹果（生态+品牌）、腾讯（社交网络效应）相比，装机量护城河太容易被政策击穿。在集采面前，"装了4万台机器"≈零议价能力。

> **④ 飞轮被打断。** "装仪器→卖试剂→赚钱→研发→装更多仪器"的飞轮，集采砍掉一半试剂利润后转速大降。这是结构性问题，不是周期性问题。

> **⑤ 安全边际不够。** 段永平买点通常需要30%+安全边际。现价42.39元对应ROE=18%，集采后大概率15%→公允价36元→至少要跌到30元以下（PE~14x）才可能进入观察区。

### 8.4 什么情况下会考虑买入？

```
情景A：股价跌到25元以下（PE 12x，PB 2.1x）
  市场过度悲观，price in ROE=10%极端预期
  实际ROE大概率不低于12-15%
  存在明显"悲观定价错误" → 段永平可能"看一看"

情景B：集采政策明朗化，证明净利率稳定在30%+
  不确定性消除，可预测性提升
  此时即使股价回升也可能考虑

情景C：海外业务收入占比超过50%
  变成"全球IVD公司"，国内集采影响显著稀释
  估值从政策驱动转为成长驱动
```
