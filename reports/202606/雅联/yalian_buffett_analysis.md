## 一、生意模式分析

### 1.1 一句话理解

亚联机械做的是**人造板生产线设备**——就是把木屑、刨花压成中密度纤维板（MDF）和刨花板（PB）的整条生产线。客户是板材厂（如大亚圣象、丰林集团等），卖一套设备几千万到上亿不等。

```
木屑/刨花 → 亚联的生产线设备 → MDF/刨花板 → 家具厂/地板厂
            ↑
       亚联的位置：卖"生产板材的机器"
```

## 二、护城河分析

### 2.1 五维评估

| 护城河类型 | 强度 | 详细论证 |
|:---|:---|:---|
| **转换成本** | ⭐⭐ 弱 | 生产线设备是大额一次性采购，客户在下一次扩产时完全可以换供应商 |
| **技术壁垒** | ⭐⭐ 弱 | 人造板设备是成熟技术，最核心的热压机/铺装机国内多家能做 |
| **品牌** | ⭐⭐ 弱 | 有一定行业知名度，但设备采购以技术参数和价格为主 |
| **成本优势** | ⭐ 不存在 | 毛利率仅14%，说明没有成本优势——甚至可能在亏本抢单 |
| **网络效应** | ⭐ 不存在 | 设备制造商，一锤子买卖 |

### 2.2 ⚠️ 三个致命信号

| 信号 | 数据 | 说明 |
|:---|:---|:---|
| **毛利率崩塌** | 25%→14% | 竞争激烈→被迫降价→利润被蚕食 |
| **经营现金流连负3年** | -1.46/-0.72/-0.32亿 | 利润全是"纸面利润"，真金白银在流失 |
| **应收款畸高** | 9.82亿(占营收60%) | 设备卖出去了，钱没收回来 |

## 四、行业前景：在夕阳里挣扎

人造板设备行业的客户——板材厂——面临：
- 房地产下行→家具需求萎缩→板材需求减少
- 环保政策趋严→中小板材厂关停→设备采购需求萎缩
- 产能过剩→客户不扩产→没有新设备订单

> 亚联面对的不仅是一个竞争激烈的市场，更是一个**需求在萎缩的市场。**
