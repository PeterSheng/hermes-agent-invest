# Lululemon 长期趋势分析：李录视角

> **分析日期**：2026年7月8日
> **股价**：$115.07 | **市值**：~$131亿 | **PE (TTM)**：9.59x
> **核心命题**：Lululemon 是文明级趋势(athleisure)的受益者，还是这个趋势拐点的受害者？
>
> *「价值投资的核心不是买'好公司'，而是以远低于内在价值的价格买入你能真正看懂的公司。」—— 李录*

---

## 序言：李录为什么会关注 Lululemon？

李录是查理·芒格的合伙人、喜马拉雅资本创始人。他的投资框架有几个鲜明特征：

1. **从文明演进的大尺度看问题**——他投资比亚迪，看的是化石燃料向清洁能源的文明级转型；他投资茅台，看的是中国消费升级的30年级别趋势。

2. **极度重视管理层**——"创始人精神"是他最看重的东西之一。职业经理人可以把好公司管成一般公司，而创始人可以让一般公司变成伟大公司。

3. **中美双栖的独特视角**——作为少数同时深刻理解中美市场的投资者，他能看到单一文化视角下的盲点。

4. **极度诚实的能力圈**——他说过："我95%的时间都在说'我不懂'。"他宁可错过100个机会，也不碰一个"以为自己懂但其实不懂"的东西。

用这套框架审视 Lululemon，会得出与其他三位大师截然不同但高度互补的结论。

## 一、文明级趋势：Athleisure 浪潮到了拐点吗？

### 1.1 李录的"大趋势"分析框架

李录投资比亚迪的逻辑不是"电动车是个好生意"，而是"人类从化石燃料向清洁能源的转型是300年级别的文明演进"。他用类似的框架看消费品：

> **一个消费品品牌能否成为"百年老店"，取决于它是否绑定了人类文明进程中不可逆的深层需求变迁，而非短期时尚潮流。**

### 1.2 Athleisure 的本质：三大文明级变迁的交叉点

李录会把 athleisure 拆解为三个层级的趋势，每一层有不同的生命周期：

| 层级 | 趋势 | 文明驱动力 | 生命周期 | 对 Lululemon 的含义 |
|:---|:---|:---|:---|:---|
| **L1: 健康意识觉醒** | 全球中产以上人群从"不运动"到"规律运动" | 后工业社会人类寿命延长 + 慢性病危机 + 知识工作者的身体焦虑 | **50-100年级别，不可逆** | 基本面支撑，瑜伽/健身人群只会越来越多 |
| **L2: 着装规范松弛** | 从"西装革履"到"运动休闲日常化" | 硅谷科技文化的全球扩散 + 远程办公常态化 + 身份信号从"正式"转向"自律" | **20-30年，仍在中期** | 运动服日常化不是时尚，是文明变迁——但这不等于"买$128的瑜伽裤"是必然 |
| **L3: 瑜伽文化符号化** | 瑜伽从中产阶级灵修变成大众消费符号 | 社交媒体加速的"健康人设"表演 + 女性消费力上升 | **10-15年，可能接近拐点** | ⚠️ 这是 Lululemon 最依赖的层级，也是风险最大的层级 |

### 1.3 类比：星巴克在中国的"第三空间"范式

李录对中国消费品的理解极其深刻。他会把 Lululemon 与星巴克在中国的经历进行类比：

| 维度 | 星巴克（中国，1999-2024） | Lululemon（全球，2010s-2020s） |
|:---|:---|:---|
| **核心叙事** | "第三空间"——中产阶级生活方式的符号 | "运动身份的社交货币"——健康/自律的信号 |
| **溢价逻辑** | ¥35 一杯咖啡 = 同时买了一杯饮品 + 两小时社交空间使用权 | $128 一条裤子 = 同时买了面料 + 身份信号 |
| **范式基石** | 中国城市化 + 白领阶层扩大 | 全球瑜伽/健身文化 + 社交媒体的"健康人设" |
| **范式何时破裂** | 当瑞幸用¥9.9证明"第三空间不值¥25溢价"，当中国本土咖啡文化成熟 | 当Alo/Vuori证明"运动身份信号不值$50溢价"，当消费者审美疲劳 |
| **当前状态** | 星巴克中国仍在增长但增速骤降，单店收入下滑，被迫加入价格战 | ⚠️ Lululemon 美洲同店转负，可能正在经历星巴克式的"范式侵蚀" |

**李录的核心判断**：

> **Athleisure 作为一种"文明现象"远未结束——人们不会重新穿回西装。但 Lululemon 作为这一趋势的"最大受益者"可能已经过了巅峰。文明趋势的生命周期 ≠ 品牌的生命周期。就像"喝咖啡"这个行为会持续500年，但星巴克不一定能再辉煌50年。**

### 1.4 拐点信号——李录会关注的四个文明级指标

| 指标 | 观察 | 结论 |
|:---|:---|:---|
| **社交媒体话语** | Gen Z 在 TikTok/小红书上的语言体系：Lululemon 是"basic"还是"iconic"？ | ⚠️ "妈妈裤""校服"等标签化话语在加速蔓延——这是文化地位下降的早期信号 |
| **竞品的文化叙事力** | Alo Yoga = "好莱坞酷"，Vuori = "加州自在"，Skims = "身体包容" | 🔴 Lululemon 的叙事停留在"瑜伽+社区"——这在2018年是领先的，在2026年已经陈旧 |
| **替代性身份信号** | 后疫情"报复性打扮"——西装、连衣裙、牛仔裤回归 | 🟡 不是威胁 L1(健康意识)，是威胁 L3(瑜伽裤作为社交货币) |
| **定价权的社会心理基础** | $128 瑜伽裤 vs $30 平替——消费者认知差距在缩小 | 🔴 TikTok 的平替测评正在系统性消解溢价认知 |

## 二、管理层深度评估——李录最看重的东西

### 2.1 李录的管理层分析框架

李录曾在演讲中说："我宁愿投资一家二流生意里的一流管理层，也不投一流生意里的二流管理层。"他的管理层四维评估：

```
┌──────────────────────────────────────────┐
│          李录管理层四维评估模型            │
├──────────────────────────────────────────┤
│ 1. 是否具备"创始人精神"？                  │
│    → owner-operator vs professional manager│
│ 2. 资本配置能力如何？                      │
│    → 回购时机、并购决策、资本纪律           │
│ 3. 能否认识并应对范式转变？                 │
│    → 当增长模式从"开店"切换到"产品"时       │
│ 4. 团队稳定性和文化传承                     │
│    → 关键人物流失 = 灵魂流失               │
└──────────────────────────────────────────┘
```

### 2.2 Calvin McDonald：深度画像

| 维度 | 详细评估 |
|:---|:---|
| **职业履历** | 2013-2018: Sephora Americas CEO（美妆零售）。此前在 Loblaw（加拿大最大零售商）任高管。**零运动品牌经验，零服装制造经验。** |
| **核心能力** | 零售运营 + 门店扩张 + 会员体系（Sephora 的 Beauty Insider 是业内标杆之一） |
| **在 Lululemon 的成绩** | 2018年接手，营收从 $33亿→$96亿（5年CAGR 24%），门店从 440→700+，推出男装和鞋履 |
| **核心短板** | **产品创新能力未有证据证明**。Sun Choe 负责的产品创新才是真正的增长引擎 |
| **"Power of Three ×2"计划** | 2022年提出：到2026年营收$125亿（男装翻倍、DTC翻倍、国际市场翻四倍）。——当前进展远落后于计划 |

**李录的关键判断**：

> Calvin McDonald 是一个**优秀的职业经理人，但不是一个"创始人型CEO"**。他的整个职业生涯都在"管理增长"——把已经被验证的商业模式通过开店和渠道扩张进行规模化。Sephora 从 500 家开到 1000 家，Lululemon 从 440 家开到 700 家，本质上是同一套技能。
>
> **但 Lululemon 现在需要的不是"如何开更多店"，而是"如何在不开店的存量市场里让消费者继续选择 Lululemon"**。这需要产品创新力、品牌叙事力、文化敏感度——这些都不是 Calvin 的核心能力圈。
>
> **更关键的是**：李录投比亚迪时，王传福是工程师出身的技术狂人——他能亲自设计电池、想清楚电车产业链的每个环节。Calvin McDonald 能亲自设计一条瑜伽裤吗？能判断下一季的面料趋势吗？

### 2.3 Sun Choe 离职——李录眼中的"红旗中的红旗"

| 维度 | 分析 |
|:---|:---|
| **她在 Lululemon 的角色** | 2016年加入，2018年升任 CPO。直接负责的产品线包括：Align 系列（公司最大现金牛）、Define Jacket（时尚转型单品）、男装线扩张、鞋履从0到1 |
| **她的履历质量** | Urban Outfitters、Madewell、Marc by Marc Jacobs——**纯正的"产品/品牌"背景**，与 Calvin 的"零售/运营"背景形成互补 |
| **离职后去向** | VF Corporation（Vans、The North Face 母公司）——全球品牌总裁。注意：不是去了一家初创公司，而是一家急需产品创新的大集团。这说明她认为 Lululemon 的产品创新空间不如另一家大公司 |
| **组织替代方案** | Lululemon 宣布**不再设置 CPO 职位**，由 CEO 直接管理产品团队。这等于说："我们找不到替代者，或者我们不认为这个角色值得保留。" |
| **芒格已警告的时间窗口** | 产品管线滞后约12-18个月。Sun Choe 2024年5月离职 → 她最后参与设计的产品在2025年底-2026年中售罄 → **当前2026年7月正是产品断层开始显现的窗口** |

**李录的终极拷问**：

> 当一个公司的产品灵魂人物离开，而 CEO 说"我来直接管产品"——在一个他没有任何产品设计经验的行业——这不是"领导力"，这是**傲慢**。真正伟大的CEO知道自己不懂什么。

### 2.4 管理层资本配置评估

| 决策 | 评级 | 分析 |
|:---|:---:|:---|
| **零有息负债** | ✅ 极好 | 不借债——这是巴菲特/芒格/李录都推崇的资本纪律 |
| **拒绝分红** | ✅ 正确 | 成长型企业不应分红，应保留资金再投资 |
| **零并购** | ✅ 正确 | 没有做愚蠢的收购（对比 Nike 收购 Converse） |
| **股票回购** | 🔴 灾难级 | 在 $250-$380 均价回购了 ~$40亿，当前股价$115。这要么是极度糟糕的时机判断，要么是用股东的钱给高管期权提供掩护 |
| **CAPEX投中国** | 🟡 中性偏正 | 中国是唯一有大规模增量空间的市场——但这个赌注尚未验证 |

> **李录的评价**：「在 $300 均价回购$40亿，是过去五年消费品公司最糟糕的资本配置案例之一。如果那些钱留在账上，现在可以在$115回购双倍股份。这不是'判断失误'——这是缺乏资本配置纪律的直接证据。」

## 三、中美消费品牌生命周期——为什么这是李录的独特优势

### 3.1 美国运动品牌的平均寿命远长于中国品牌——原因分析

这是李录能提供独特洞见的核心话题。他在中美的深度生活经验让他能看到两个市场品牌生命周期的根本差异：

| 维度 | 美国市场 | 中国市场 | 对 Lululemon 的含义 |
|:---|:---|:---|:---|
| **品牌生命周期** | 长青品牌多（Nike 58年、Vans 57年、Converse 115年、Levi's 170年） | 品牌迭代快（李宁34年已是"老牌"，回力翻红又翻黑，完美日记5年巅峰） | Lululemon在美国有更长尾的生存空间 |
| **核心原因** | ①品牌是"代际传承"的文化资产（父母穿Nike→孩子穿Nike）；②零售渠道成熟稳定；③消费者品牌忠诚度高 | ①品牌是"当下认同"的社交工具；②零售/电商渠道每3年一洗牌；③消费者极度追逐"新" | Lululemon在中国面临更短的品牌半衰期 |
| **失败模式** | 缓慢衰退（Gap 从巅峰到谷底用了15年） | 断崖式下跌（喜茶从排队7小时到门可罗雀用了2年） | 中国一旦失速，比美国更可怕 |
| **国潮替代效应** | 弱——美国没有"穿美国品牌=爱国"的叙事 | 强——"新疆棉"让Nike一夜跌15%份额，李宁/安踏吃掉了溢出的需求 | 🔴 **这是Lululemon中国的最大单一风险** |

### 3.2 Lululemon 在品牌生命周期中的位置

```
品牌生命周期曲线（李录修正版，区分中美两条曲线）：

美国：
  诞生(1998) ──→ 小众(2000s) ──→ 爆发(2014-2022) ──→ ⚠️当前：普及→衰退的交叉口
                    ↑                              ↑
               Chip Wilson时代           Calvin McDonald时代
                                          (渠道扩张红利)

中国：
  进入(2016) ──→ 小众(2018) ──→ 爆发(2020-2024) ──→ ⚠️当前：接近巅峰，面临"国潮替代"临界点
                                        ↑
                              "中产女性身份标签"的叙事顶峰
```

**美国市场**：Lululemon 可能正从"普及期"滑向"过气期"。这不是说它会消失——Gap 也没消失——而是说它的"酷度溢价"正在蒸发。

**中国市场**：Lululemon 仍处于"爆发的尾巴"，但中国品牌的加速周期远快于美国。在美国，一个品牌从"酷"到"不酷"可能需要5-10年；在中国，可能只需要2-3年。

### 3.3 李录最关心的文化信号

| 信号 | 美国 | 中国 |
|:---|:---|:---|
| **"穿 Lululemon 代表什么"的文化含义** | 从"自律的瑜伽爱好者" → "郊区中产妈妈" | 从"国际精英" → "有钱但不懂本地品牌" |
| **替代选择的文化叙事** | Alo Yoga = "我在好莱坞有个朋友" Vuori = "我冲浪，不是只做瑜伽" | MAIA ACTIVE = "我支持国货也懂身材" 粒子狂热 = "我是新一代中国女性" |
| **代际信号** | Gen Z 转向 Alo/Skims——Lululemon 正在变成"Millennial 品牌" | Gen Z 从一开始就没有把 Lululemon 当作"自己的品牌" |

## 四、中国市场——第二增长曲线还是下一个 Nike 陷阱？

### 4.1 Nike 在中国的前车之鉴——李录会讲的警示故事

Nike 中国在 2021-2024 年发生了什么，是每个研究 Lululemon 中国的人都应该深度学习的历史：

| 时间 | Nike 中国 | Lululemon 中国的平行线 |
|:---|:---|:---|
| **2015-2019**: 黄金时代 | 大中华区营收从 $3B→$7B，年增20%+，无人能敌 | **等价当前**: Lululemon 中国门店从50→130家，增速30%+ |
| **2020-2021**: 转折点 | 疫情福利消退 + 新疆棉事件 → 品牌首次在中国遭遇"爱国替代" | **潜在触发**: 中美关系紧张升级 → 消费者转向"国潮瑜伽" |
| **2022-2024**: 崩塌期 | 安踏营收超越Nike中国，李宁从"过气"翻红，Nike开始打折 → 品牌高端形象受损 | **⚠️ 如果触发**: MAIA ACTIVE（安踏旗下）、粒子狂热将系统性接收溢出的需求 |
| **当前**: 僵局 | Nike 中国仍在赚钱，但增长故事结束了。它从"中国第一运动品牌"变成了"外国品牌之一" | **Lululemon可能的命运**: 从"瑜伽服= Lululemon"变成"lululemon是瑜伽服品牌之一" |

### 4.2 中国市场的真实天花板

李录会用数据说话：

| 指标 | 美国 | 中国 | 差距分析 |
|:---|:---|:---|:---|
| **瑜伽人口渗透率** | ~10-12%（约3300万人规律练习） | ~1-2%（约1500-3000万人） | 中国瑜伽人口增长空间大 |
| **高端运动服TAM** | ~$40B（成熟市场） | ~$8-12B（快速增长但容量有限） | 中国市场天花板远低于美国 |
| **Lululemon 门店密度** | ~370家（3.3亿人）= 1店/90万人 | ~130家（14亿人）= 1店/1077万人 | 渗透空间大，但中国地域差异极大 |
| **人均运动服支出（高端）** | ~$120/年/高收入人群 | ~$25-35/年/高收入人群 | 付费意愿差异巨大 |
| **价格承受力** | $128 = 中产日工资的 1/3-1/2 | ¥850 = 中产日工资的 50%-80% | 价格天花板更硬 |

**核心矛盾**：

> Lululemon 在中国的问题不是"瑜伽人口不够多"，而是：
> 1. **愿意花 ¥850 买一条裤子的人数增长，赶不上 Lululemon 开店的速度**
> 2. **本土品牌的性价比优势（¥300）正在从"便宜替代"变成"更好的选择"**
> 3. **中国消费者从"仰望国际品牌"到"平视所有品牌"的心态转变不可逆**

### 4.3 李录的终极判断

> 「中国市场对 Lululemon 来说，是一个**真实的、但被高估的增长期权**。它可以支撑 3-5 年的门店扩张增长，但它不会成为'下一个美国'——不是因为市场不够大，而是因为竞争格局和文化根基完全不同。美国是 Lululemon 的'家乡市场'，有文化根基；中国永远是 Lululemon 的'客场市场'，每一次地缘政治风波都是潜在的品牌灾难。」
>
> **更重要的**：如果美国市场继续恶化，中国市场的增长根本无法弥补。美洲 $6B+ 的盘子掉 5% = 亏 $300M，中国 $1.5B 的盘子涨 20% = 只增加 $300M。数学上是平的——但前提是美洲只掉 5%，而 Q1 信号暗示可能掉得更多。

## 五、百年老店基因——Lululemon 能不能穿越周期？

### 5.1 李录的"百年老店"基因清单

李录的投资组合中有两类"接近百年老店"的公司：**茅台**（中华文明的酒精符号）和**比亚迪**（新能源文明的基础设施）。他会用以下框架检验 Lululemon：

| 百年老店基因 | 茅台 | 爱马仕 | Nike | Lululemon | 缺失程度 |
|:---|:---:|:---:|:---:|:---:|:---:|
| **1. 绑定基础人类需求** | ✅ 酒精/社交 | ✅ 身份展示 | ✅ 运动/身份 | ⚠️ 健康意识(L1强)+身份信号(L3弱) | 部分满足 |
| **2. 不可复制性** | ✅ 茅台镇微生物环境 | ✅ 180年代际传承的手工体系 | ⚠️ 品牌+运动员矩阵 | ❌ 面料可代工，设计可复制 | **严重缺失** |
| **3. 定价权随时间增长** | ✅ 越陈越贵 | ✅ Birkin永远涨价 | ⚠️ 需要持续创新维持 | ❌ 定价权随品牌"酷度"波动 | **严重缺失** |
| **4. 文化根基的深度** | ✅ 5000年白酒文明 | ✅ 欧洲奢侈品文明 | ✅ 50年+体育文化 | ❌ 25年，绑定的是"时尚趋势" | **严重缺失** |
| **5. 代际传承能力** | ✅ 爸爸喝茅台→儿子喝茅台 | ✅ 妈妈背Birkin→女儿背Birkin | ⚠️ 部分成立 | ❌ Gen Z主动远离 | **严重缺失** |
| **6. 管理层传承机制** | ✅ 国企体系，不依赖个人 | ✅ 家族+工匠体系 | ⚠️ 职业经理人体系 | ❌ 创始人出走+CPO离职，无传承 | **严重缺失** |
| **7. 反脆弱性** | ✅ 经济越差越喝 | ✅ 富人越有钱越买 | ⚠️ 经济下行承压 | ❌ 完全暴露于消费者情绪 | **严重缺失** |

### 5.2 为什么 Lululemon 不是"运动界的爱马仕"

许多投资者喜欢这个类比，但李录会直接戳破：

| 爱马仕 | Lululemon |
|:---|:---|
| Birkin包可以传给女儿，30年后价值可能翻倍 | $128瑜伽裤穿3年后起球，二手价值$15 |
| 工匠体系——每个包由一个工匠手工完成，6代传承 | 代工厂量产——越南/孟加拉的面料和缝纫工 |
| 稀缺性是被体系保护的——年产量严格受控 | 稀缺性纯靠品牌——一旦品牌褪色，$128就变成了$128的问题 |
| 定价权是**永恒的、递增的**——Birkin每年涨价5-10% | 定价权是**暂时的、递减的**——折扣正在从季节性变常态化 |
| 没有"爱马仕平替"这个概念——平替不是爱马仕 | "Lululemon平替"是TikTok上最热门的种草话题之一 |

> **李录的评价**：「把 Lululemon 比作爱马仕，是对价值投资最基本概念的误解。爱马仕的价值建立在**不可复制的稀缺性**之上，Lululemon 的价值建立在**可复制的品牌势能**之上。前者随时间增长，后者随时间衰减。这是两种完全不同的生意。」

### 5.3 当前基因评分

| 基因 | 现有程度 | 能否建立？ | 时间窗口 |
|:---|:---:|:---:|:---|
| 绑定的基础人类需求 | 40% | 可以强化——转向"健康生活方式品牌"而非"瑜伽时尚品牌" | 3-5年 |
| 不可复制性 | 10% | 极难——消费品行业几乎不存在不可复制的壁垒 | N/A |
| 定价权 | 正在衰减(从90→70%) | 需要产品创新回归 + 品牌重塑 | 1-3年 |
| 文化根基 | 25% | 如果成功转型为"健康生活方式"，可能建立30-40年根基 | 5-10年 |
| 代际传承 | 5% | 需要Gen Z重新拥抱——目前方向相反 | 5年+ |
| 管理层传承 | 15% | 需要从外部引入"产品狂人"型CEO | 即刻 |

> **综合判断**: Lululemon 目前具备的"百年老店"基因不超过 **20%**。它更像一个典型的"20-30年级别"的消费品牌——不是昙花一现（品牌基础比Under Armour好得多），但缺乏穿越世纪的内核。

## 六、能力圈诚实——李录最重要的原则

### 6.1 Lululemon 在李录的能力圈吗？

**答案：不在核心圈，但在边缘可触达的区域。**

李录的能力圈特征：
| 能力圈层级 | 典型标的 | Lululemon 的位置 |
|:---|:---|:---|
| **核心圈**（深入研究+重仓持有） | 比亚迪、茅台、美光科技（限时） | ❌ 不在 |
| **理解圈**（能看懂但不一定有深入洞见） | 苹果、Costco、某些中国消费品牌 | ⚠️ Lululemon 在此 |
| **观察圈**（持续追踪但不下注） | 特斯拉、Nike、星巴克 | Lululemon 更接近此 |
| **完全不懂圈** | 生物科技、加密货币 | — |

### 6.2 为什么 Lululemon 比茅台/爱马仕更难看懂？

这是李录视角最诚实也最有价值的部分：

| 茅台 | Lululemon |
|:---|:---|
| 需求端：**酒精是人性中的常数**——5000年来不变 | 需求端：**"穿瑜伽裤很酷"是社会建构**——可能在5年内改变 |
| 供给端：**茅台镇的微生物环境是物理常数**——不可复制 | 供给端：**面料+设计+营销是商业变量**——任何人可以复制 |
| 竞争格局：**茅台没有真正的竞争者**（酱香型高端白酒=茅台） | 竞争格局：**每周都有新竞争者**（Alo/Vuori/Skims/MAIA/安踏/...） |
| 定价权：**永恒的**——涨价=销量不变 | 定价权：**有条件的**——涨价可能加速客户流失 |
| 文化根基：**5000年白酒文明**，无需"维持热度" | 文化根基：**25年运动时尚**，必须不断"维持热度" |
| 管理层重要性：**中等**——即使管理层一般，茅台也能活得很好 | 管理层重要性：**极高**——品牌热度全赖管理层维持 |
| 可预测性：**极高**——10年后茅台财报偏差不超过20% | 可预测性：**极低**——10年后 Lululemon 有可能翻5倍也可能跌80% |

> **李录会说**：
>
> 「投资茅台，我只需要回答一个问题：10年后中国人还喝不喝白酒？答案是几乎确定的。
>
> 投资 Lululemon，我需要回答至少五个问题：
> 1. 10年后瑜伽裤还流行吗？
> 2. 10年后 Lululemon 还酷吗？
> 3. 10年后它的竞争者是谁？
> 4. 10年后它的管理层还是这群人吗？
> 5. 10年后的定价权还剩多少？
>
> 这五个问题中的任何一个我都无法给出高确定性的答案。而在我的投资框架里，**如果一个投资的确定性不够高，不管潜在收益有多大，它都不属于我的游戏。**」

## 八、李录式终判

> *「如果我是在1998年遇到 Chip Wilson 时期的 Lululemon——一个瑜伽爱好者在温哥华用自己对面料的理解创造了一个新品类——我会对这个创始人和他的公司产生极大的兴趣。因为那是一个'创始人用热爱创造价值'的故事。*
>
> *但在2026年，我看到的是一个职业经理人管理的、增长模式已经耗尽的、品牌热度正在消退的、竞争对手四面围攻的零售公司。PE 9.59× 反映的是市场对'增长消失'的恐惧，而这种恐惧——在这个案例中——并非不合理。*
>
> *在喜马拉雅资本，我们有一条铁律：永远不要把你的钱投进一个你无法回答'10年后这家公司会怎样'的公司。对于 Lululemon，我诚实地回答：我不知道。*
>
> *而'我不知道'——在价值投资的词典里——永远翻译为'不买'。」*
