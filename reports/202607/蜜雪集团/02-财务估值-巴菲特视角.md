## 二、DuPont ROE 拆解（2025年）

```
ROE = 37.0%

  =  净利率      ×  资产周转率    ×  权益乘数
     17.7%          1.40x           1.50x

     高              高              中低
```

| 组件 | 数值 | 行业对比 | 诊断 |
|:---|:---|:---|:---|
| 净利率 | 17.7% | 行业平均~8-12% | ✅ 优秀（供应链自产拉高利润） |
| 资产周转率 | 1.40x | 零售/餐饮~1.5-2.5x | ⚠️ 中等（轻资产模式应该更高） |
| 权益乘数 | 1.50x | 安全线<3.0x | ✅ 低杠杆，财务稳健 |

> 🔑 **ROE 37% 的质量判断**：高净利率来自真实竞争优势（自产降本），不是靠加杠杆。这是一台**高质量高ROE的复利机器**。

### ROE 趋势：40.5% → 37.0%（"好下降"还是"坏下降"？）

2025年ROE从40.5%降至37.0%，原因是 **IPO融资增加了权益基数**（净资产从110亿→160亿），而利润增速（+32.7%）低于净资产增速（+45%）。这属于**"好下降"**——资金在手，等待投入再生产。如果ROE继续下降而利润不再增长，才会变成"坏下降"。

## 四、估值框架

| 估值指标 | 数值 | 巴菲特标准 | 判断 |
|:---|:---|:---|:---|
| **PE (TTM)** | **14.9x** | <15x = 合理 | ✅ 合理偏低 |
| **PB** | **5.5x** | 取决于ROE | ⚠️ 偏贵（但ROE 37%可解释） |
| **PEG** | **0.65x** | <1.0 = 低估 | ✅ 显著低估 |
| **FCF Yield** | 5.7% | >5% = 有吸引力 | ✅ |
| **隐含回报率 (1/PE)** | 6.7% | — | 高于无风险利率 |

### PE-ROE 性价比矩阵

```
PE = 14.9x，ROE = 37%
→ 每1元利润的价格 = 14.9元，但这1元利润能生出37%的回报
→ PE/ROE = 0.40（越低越好，<1.0即为合理）
→ 对比：古茗PE~20x/ROE~25% = 0.80，蜜雪性价比更优
```

## 六、巴菲特式结论

> **蜜雪冰城的数字让我想起1972年的See's Candies——它有一项别人无法复制的经济特征：极低的成本结构。37%的ROE搭配14.9倍的PE，PEG只有0.65，FCF Yield达到5.7%。如果它能再保持这个ROE十年，现在的价格就是送礼。**
>
> **但我必须承认：我对估值模型里的假设不太舒服。12%的FCF增长隐含了海外扩张的成功和新品类（咖啡/啤酒）的突破——这两件事的确定性远不如它卖柠檬水。我宁愿用更保守的数字：如果FCF只增长8%，内在价值只有300 HKD，安全边际18%——那就不那么诱人了。**
>
> **最大的风险藏在资产负债表之外：6万家加盟商中，有多少在亏钱？他们能撑多久？蜜雪不承担门店亏损，但它承担"加盟商集体关店"的后果。这个风险，财务报表上找不到。**

| 巴菲特综合评分 | **3.75 / 5.0** ★★★¾ |
