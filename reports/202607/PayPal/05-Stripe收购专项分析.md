# 05 Stripe收购PayPal专项分析

> **"一家$1,590亿的私人公司 + 一家PE巨头，试图用$530亿买下一家$400亿的上市公司。这是捡漏还是战略赌博？"**

---

## 一、交易概要

| 项目 | 详情 |
|------|------|
| **收购方** | Stripe + Advent International |
| **目标** | PayPal Holdings (PYPL) |
| **报价** | $60.50/股，总价 ~$530亿 |
| **溢价** | 较收购前$47.27溢价28% |
| **融资** | ~$500亿银行融资支持 |
| **时间线** | 4月首次接洽 → 7月正式报价 |
| **竞标方** | Block (Square) 可能参与 |

---

## 二、战略逻辑：这笔交易为什么"成立"？

### Stripe要什么？

| PayPal资产 | 对Stripe的价值 |
|-----------|---------------|
| **439M消费者账户** | Stripe只有商户端，缺消费者触达——这是最大的互补 |
| **Venmo 113M用户** | 年轻人的社交支付图谱——这是Stripe用钱买不来的 |
| **Branded Checkout** | 消费者在商家网站上看到"Pay with PayPal"可以直接支付——完成商户+消费者闭环 |
| **加密/稳定币牌照** | PayPal已整合Paxos比特币服务，有50州的MTL牌照 |
| **Braintree** | PayPal的商户收单业务——可以直接整合到Stripe平台 |

### Advent要什么？

| 要素 | 分析 |
|------|------|
| **稳定现金流** | PayPal FCF $65亿/年——PE基金最喜欢的"现金牛" |
| **成本削减空间** | 裁员20%只是开始，Advent可以把PayPal从23,800人砍到15,000人 |
| **退出路径** | 整合3-5年后重新IPO，估值$1,000-1,500亿 |

### 合并后的图景

```
合并实体: Stripe + PayPal
├── 商户端: Stripe (5M商户, $1.9T TPV)
├── 消费者端: PayPal + Venmo (439M账户, $1.79T TPV)
├── 合计TPV: $3.7万亿
├── 合计营收: ~$65B (估)
└── 加密: Bridge + Paxos = 全栈加密支付
```

**逻辑自洽度：★★★★☆**

## 三、价格分析：$60.50是公平价格吗？

| 估值锚 | 对应股价 | vs $60.50 |
|--------|:-------:|:--------:|
| **收购前市场价** | $47.27 | +28% |
| **2021年ATH** | $308.00 | **-80%** |
| **DCF悲观情景** | $63 | +4% |
| **DCF基准情景** | **$85** | **-29%** |
| **DCF乐观情景** | $115 | -47% |
| **同行对比PE (Block ~20x)** | ~$106 | -43% |

### 结论：$60.50对Stripe是"白菜价"，对PayPal股东是"低价逼仓"

- **从Stripe角度**：11.3倍PE买一个439M用户的支付平台——"这是十年一遇的捡漏机会"
- **从PayPal股东角度**：$60.50比内在价值（$85）低29%，比同行估值低43%——"被贱卖了"
- **但从另一个角度看**：如果PayPal自己不争气，3年后EPS跌到$4.00，$60.50反而是"及时止损"

## 四、成功概率评估

| 因素 | 概率 | 分析 |
|------|:----:|------|
| ✅ 战略匹配度 | **75%** | 商户+消费者 = 完美闭环 |
| ✅ 融资可行性 | **70%** | $500亿融资已到位 |
| ⚠️ 价格充分性 | **55%** | 28%溢价偏低，董事会可能要求$70-75 |
| 🔴 反垄断 | **40%** | 合并后线上支付市占率超50%，审查风险极高 |
| 🔴 文化整合 | **35%** | 8K精英 vs 24K大公司——"油和水" |
| ⚠️ 竞标风险 | **50%** | Block可能出价$65-70搅局 |
| ⚠️ PayPal独立性 | **45%** | 新CEO Lores可能说服董事会拒绝 |

### 加权成功概率：**~50%**

### 最可能的5种结果：

| 概率 | 结果 |
|:----:|------|
| **30%** | 提价至$68-72 + 反垄断让步 → 交易完成 |
| **25%** | 反垄断否决 → 交易破裂 → 股价跌回$40 |
| **20%** | PayPal拒绝，坚持独立转型 |
| **15%** | Block加入竞标战，最终价$75+ |
| **10%** | 其他变数（股东诉讼、融资失败等） |
