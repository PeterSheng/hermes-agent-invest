## 二、DuPont ROE 拆解

伯克希尔的 ROE 不能用传统 DuPont 框架——它的盈利中包含大量"未实现投资收益"，波动极大。拆开来看运营层面：

| 指标 | 近似值 | 评判 |
|:---|:---:|:---|
| 运营 ROE（剔除投资波动） | ~8-10% | 尚可，但不算惊艳 |
| 净利率（运营层面） | ~12-15% | 分散化业务拉低整体利润率 |
| 资产周转率 | ~0.35x | 极低——这是重资产控股公司的宿命 |
| 权益乘数 | ~1.5x | 极其保守，几乎零杠杆 |

> **值得注意的是**：ROE 的"低"不是坏事。伯克希尔主动选择了去杠杆+囤现金，这是一种质量而非风险的下降。

## 三、现金流质量——巴菲特最看重的指标

| 指标 | 数据 | 评判 |
|:---|:---|:---|
| 运营现金流 / 净利润 | > 1.3x | ✅ 利润含金量极高 |
| 自由现金流 | 巨额，但大部分回流到投资组合 | ⚠️ "自由现金流"在伯克希尔是个模糊概念 |
| 分红 | 零 | 巴菲特认为他能比股东更好地配置资本 |
| 回购 | PB > 1.2x 时暂停 | 巴菲特的纪律在起作用 |

伯克希尔的特殊之处在于：它的大部分"资产"是上市股票，有实时市值——这让传统现金流分析的意义大打折扣。**真正重要的是资本配置决策的质量，而不是现金流数字本身。**

## 四、估值——Buffett Indicator 视角

巴菲特本人说过，伯克希尔的"内在价值"可以用市值/账面价值粗略衡量。他曾在 PB ≤ 1.2 时大规模回购。

### 核心估值指标

| 指标 | 数值 | 评判 |
|:---|:---:|:---|
| 当前 PB | **~1.5x** | 高于巴菲特回购门槛（1.2x） |
| 现金/市值 | **36%** | 极高——每投入 $1，有 $0.36 是现金 |
| AlphaSpread 内在价值 | **$396** | 高估 21% |
| ValueInvesting.io 内在价值 | **$472** | 高估 7% |
| AI 共识目标价 | **$517** | 基本持平 |
| Brian Maass 综合估算 | **~$430-470** | 高估 8-15% |

### 安全边际计算

```
保守内在价值（多方平均）：~$450
当前股价：$507
安全边际：(450 - 507) / 507 = -12.6%

⚠️ 无安全边际，甚至有溢价
```

### 巴菲特的"自己的股票自己买"框架

| 条件 | 当前状态 | 动作 |
|:---|:---|:---|
| PB ≤ 1.2x | ❌ 1.5x | 不应回购 |
| PE ≤ 15，ROE > 13% | ✅ PE~12，ROE~10-12% | 边界模糊 |
| 现金过多（>30%市值） | ✅ 36% | 现金拖累回报 |
| 找不到替代标的 | ✅ 已多年无大象级收购 | 该买但没有目标 |

## 五、内在价值 DCF（简化版）

```
假设（保守）：
  - 运营 FCF 增速：5%（前5年），3%（终值）
  - 折现率：9%
  - 当前运营 FCF/股：~$15

现值（前5年）：$68
终值现值：$281
内在价值/股：~$349（仅运营部分）
加投资组合/股：~$118（$3290亿 ÷ 27.8亿股）
加现金/股：~$143（$3970亿 ÷ 27.8亿股）

总计内在价值：$349 + $118 + $143 = $610/股

但注意：这是"拆开卖"的价格。作为一整个公司，应给予控股公司折扣
（~15-20%）：$610 × 0.80 = $488

vs 当前 $507 → 基本合理，无安全边际
```
