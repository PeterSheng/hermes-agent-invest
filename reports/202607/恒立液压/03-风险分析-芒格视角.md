## 芒格的第一性原理问题

> **"告诉我，这家公司怎么死？"**

如果恒立液压会崩盘，最可能的路径不是一条，而是**多条路径的叠加**。让我用逆向思维枚举所有可信的失败路径：

---

## 🔴 六大可信失败路径

### 路径1：工程机械周期深度下行
| 要素 | 分析 |
|------|------|
| **概率** | 40%（中国基建投资已过峰值，设备保有量高位） |
| **传导机制** | 挖机销量下滑→主机厂压价→恒立毛利率再降2-3个百分点→25%净利率→22%→EPS负增长 |
| **影响** | 严重。届时51倍PE会被压缩到25倍，股价腰斩 |
| **触发条件** | 地产新开工持续下滑+地方基建资金短缺 |

**历史参照**：2022年恒立净利从27亿降到23.5亿（-13%），但股价从高点跌了60%+。**周期股在周期顶部从来都不值50倍PE。**

---

### 路径2：机器人故事被证伪
| 要素 | 分析 |
|------|------|
| **概率** | 30%（人形机器人量产进度高度不确定） |
| **传导机制** | Tesla Optimus延迟量产→丝杠订单不及预期→30亿CapEx打了水漂→估值溢价瞬间归零 |
| **影响** | **致命**。当前股价至少40%是机器人期权价值。一旦证伪，PE从51x→30x，叠加利润miss，就是-50%起跳 |
| **触发条件** | Tesla推迟量产计划或转向其他供应商 |

**芒格的警告**：我见过太多"技术替代+新市场"的故事——10个里有8个以失望收场。不是说恒立的丝杠做不好，而是**从"能做"到"能大规模赚钱"之间的距离，远比华尔街想象的远。**

---

### 路径3：净利率持续压缩——"隐性商品化"
| 要素 | 分析 |
|------|------|
| **概率** | 50%（已在发生——从28.8%→25.0%，五年降了3.8个百分点） |
| **传导机制** | 下游主机厂集中度提高→议价能力增强→恒立从定价者变成价格接受者 |
| **影响** | 中高。如果净利率每年降1个百分点，增长会被完全吞噬 |

**深层问题**：恒立的油缸市占62%看似垄断，但客户三一重工一家就占了恒立营收的相当比例。**当你的客户比你更集中的时候，你的"垄断"是脆弱的。**

---

### 路径4：海外扩张失败
| 要素 | 分析 |
|------|------|
| **概率** | 25% |
| **传导机制** | 墨西哥工厂产能利用率低→海外客户认证周期超预期→地缘政治限制（中国零部件进入美国市场受限） |
| **影响** | 中高。海外收入目标50%，如果卡在30%，增长天花板被锁死 |
| **触发条件** | 中美贸易摩擦升级至液压件领域 |

恒立目前海外收入占比约13%（以资产计），但目标是50%。这中间的执行风险巨大——海外的CAT/小松用的是川崎的液压件几十年了，换供应商不是儿戏。

---

### 路径5：管理层治理风险再次爆发
| 要素 | 分析 |
|------|------|
| **概率** | 15-20%（留置原因未明，无法排除复发性） |
| **传导机制** | 汪立平再次被调查→控制权不确定性→机构抛售 |
| **影响** | 高。但如前次所示，公司运营或可正常维持 |

**芒格的思考**：我不喜欢"未知的未知"。留置事件本身只造成股价波动不到10%，说明市场认为"问题不大"。但**问题没有公开透明的结论，就意味着它可能在任何时候再次出现。** 这是治理层面的"薛定谔的猫"——放着不管可能是对的，也可能是致命疏忽。

---

### 路径6：电动化对液压的长期替代
| 要素 | 分析 |
|------|------|
| **概率** | 10-15%（10年内），30%（20年内） |
| **传导机制** | 电动缸替代液压缸→工程机械电动化→液压需求结构性下降 |
| **影响** | 低（近期）→致命（远期）。但这就是恒立做丝杠/线性驱动器的逻辑——自我替代 |

**讽刺的是**：恒立最大的"利好"（机器人丝杠）和最大的"利空"（液压被替代）是同一个故事——都是"从液压转向电驱/精密传动"。公司正在做正确的事情（自我革命），但这意味着**估值不能用"液压永续"来论证。**

## 芒格最担心的两个核心问题

### 1. 反身性陷阱：高估值→高预期→高CapEx→低FCF→估值压缩

```
当前循环：
  股价高(51x PE) → 市场预期高增长
  → 公司加大CapEx (17.4亿/年) 
  → FCF被挤压 (转化率仅9.5%)
  → 一旦增长miss → PE收缩 → 股价双杀
  → 低股价 → 融资能力下降 → CapEx被迫缩减
  → 增长真的miss → 自我实现
```

这种反身性循环在周期+成长双重属性的公司上最为危险。

### 2. "什么都想要"的战略风险

恒立同时在推进：油缸出海、泵阀升级、线性驱动器、人形机器人丝杠、液压系统集成——**五线作战**。历史上同时打赢五条战线的公司，我想不出一个。

## 芒格式结论

> **"如果一件事可能出错，它就会出错——而且通常在最坏的时刻。"**

> 恒立液压是一家**基本面优秀但被过度定价的公司**。它的危险不在于生意不好——液压件是实实在在的好生意——而在于市场把四个大概率不同时发生的好事（泵阀突破、海外扩张、丝杠放量、机器人爆发）当作"必然同时兑现"来定价。

> 我见过太多次：**市场为"最好的情景"付了"保证发生的价钱"，然后被"正常的情景"打得鼻青脸肿。** 恒立的正常情景是什么？——周期波动、净利率缓慢压缩、机器人慢于预期。在那个情景下，它值40-50元，不是104元。

> *「告诉我我会死在哪，我就永远不会去那里。」——芒格*

> 投资恒立的"会死的地方"：在51倍PE、所有人都讲机器人故事的时候冲进去。
