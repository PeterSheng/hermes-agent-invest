## 一、文明级趋势：气候变化是再保险的「风口」还是「火葬场」？

### 核心问题

> **全球气候变暖是过去50年、未来100年人类文明最重要的宏观趋势之一。再保险行业站在这个趋势的最前线——问题不是「有没有影响」，而是「方向是什么」？**

### 1.1 两种对立叙事

市场上关于「气候 × 再保险」有两套完全对立的逻辑：

| 叙事 | 逻辑链 | 隐含判断 |
|:---|:---|:---|
| **「必然受益」叙事** | 更多灾难 → 保险公司更需要再保险 → 再保险费率上涨 → 再保险公司利润爆炸 → 「气候变化的卖铲人」 | 费率弹性 > 损失弹性 |
| **「必然受害」叙事** | 更多灾难 → 损失频率和强度远超模型预测 → 精算模型失效 → 巨灾吞噬多年利润 → 「温水里的青蛙，最后被煮死」 | 损失弹性 > 费率弹性 |

**李录会怎么判断这两套叙事的真伪？他不会用「我觉得」，他会看历史数据和逻辑。** 

### 1.2 历史铁证：费率上涨永远跑不赢损失的加速度

让我们用数据回答这个问题：**过去30年，全球再保险费率上涨了多少？全球巨灾损失上涨了多少？**

| 时间周期 | 全球巨灾保险损失（年化增长率） | 再保险费率趋势 | 差距 |
|:---|:---|:---|:---|
| 1990s | ~$250亿/年平均 | 稳定偏低 | — |
| 2000s | ~$500亿/年平均 | 2005 Katrina后暴涨→2007回落 | 损失翻倍，费率回到原点 |
| 2010s | ~$900亿/年平均 | 2011日本地震+2017 Harvey/Irma/Maria→硬市场→2019软下来 | 损失再翻倍 |
| 2020-2025 | ~$1,300亿/年平均 | 2023硬市场峰值→2025已开始走软 | 损失增长远超费率增长 |

**核心数学**：
```
1990-2025，全球巨灾保险损失 CAGR ≈ 9-10%
同期再保险费率 CAGR ≈ 3-4%（即使计入硬市场峰值）
差距：损失增长是费率增长的 2.5-3倍
```

**这不是一个可以让再保险公司持续赚钱的公式。** 

### 1.3 为什么费率涨不过损失？——再保险的资本诅咒

原因很简单，而且永不可逆：**再保险行业没有护城河。**

```
灾难多发 → 损失惨重 → 再保险公司破产/收缩 →
供给减少 → 费率暴涨 → 再保险公司暴赚 →
暴赚吸引新资本涌入（ILS/对冲基金/主权基金） →
供给暴增 → 费率暴跌 → 循环
```

**每一次灾难之后，再保险公司确实能提价。但提价维持的时间越来越短，因为全球资本寻找高回报的速度越来越快。**

- 2005年Katrina后，硬市场持续了约3年
- 2017年Harvey/Irma/Maria后，硬市场持续了约2年
- 2023-2024硬市场，到2025年已经开始出现松动迹象

**资本的反应速度在加速。** 过去需要3-5年才能涌入的新资本，现在通过ILS工具6-12个月就能到位。这意味着再保险公司的「好日子窗口」正在从数年压缩到数月。

### 1.4 模型的崩塌：精算师最大的敌人不是灾难，是「变化本身」

这是芒格已经点明的核心风险，但李录会从一个更高的维度来看：

> **精算是将过去投射到未来的科学。但如果未来和过去的结构已经不同——不只是「更多」的灾难，而是「不同属性」的灾难——那么整个精算科学就成了一种精致的自欺欺人。**

具体而言：

| 气候变化的新特征 | 对再保险精算的影响 |
|:---|:---|
| **二次灾害频发**（野火、洪水、强对流） | 传统模型主要针对飓风/地震等「一次灾害」（peak peril），对二次灾害的建模严重不足。2023年全球二次灾害损失首次超过一次灾害——而多数再保险合同对二次灾害的风险敞口没有被正确定价 |
| **相关性爆炸** | 精算的核心假设是「风险相互独立」。但气候变化打破了这一点：加州野火+佛罗里达飓风+欧洲洪水可以同年同月发生（2020年的预演）。相关性意味着分散化失效 |
| **非平稳性** | 经典精算假设风险分布「平稳」（stationary）。气候变化意味着分布本身在移动——你的历史数据样本来自一个和未来不同的分布，这等于在盲飞 |
| **社会通胀 × 气候损失** | 不仅物理损失在增长，诉讼成本、赔偿标准也在增长。两个增长效应叠加——精算师对这两个变量的联合分布几乎一无所知 |

> **「再保险公司的精算师就像用1990年的交通事故数据在预测2025年的无人驾驶交通事故率——数据看起来很漂亮，但和现实没有任何关系。」**

### 1.5 芒格5星风险 + 李录文明框架 = 什么结论？

芒格从微观角度给气候风险打了5星（最高风险）。李录会从宏观角度给一个更完整的判断：

> **气候变化确实在创造巨大的再保险「需求」。但这个需求不等于利润——它等于「更多的风险需要被承载」。谁来承载？那些能在全球资本市场上获得最低成本资金的人。再保险公司在资金成本上没有优势——养老金基金只需6-7%回报率就满意，ILS投资者可以接受个位数亏损换取非相关收益。而再保险公司的股东要求15%+的ROE。** 

**水涨船高——但再保险公司不是船，它们是锚。**

| 气候变化下的赢家 | 气候变化下的输家 |
|:---|:---|
| 拥有最低资金成本的风险承载者（Berkshire的浮存金成本为负） | 依赖承保利润维持ROE的传统再保险公司 |
| ILS投资者（6-7%回报率就满意，「非相关收益」是免费的午餐） | 精算模型落后于气候变化速度的再保险公司 |
| 巨灾建模公司和数据服务商（卖铲子的人） | 在气候变化最前沿（沿海/野火区）集中承保的再保险公司 |

**Everest属于哪一类？以美国沿海飓风和加州野火为主要敞口的传统财产巨灾再保险公司——它几乎精准地站在输家那一列。**

### 1.6 气候变化终极判断

```
问题：气候变化会创造更多再保险需求吗？
答案：会。这不是问题。

真正的问题是：需求的增长能否转化为再保险公司可持续的利润增长？
答案：不能。因为每一次需求增长，供给增长得更快。
再保险的「供给」不是工厂（需要时间建），而是资本（可以在三个月内涌入）。

气候变化的真正受益者是：
  - 经纪人（Aon、Marsh McLennan）：抽水不冒险
  - 建模公司（RMS、AIR）：卖地图给淘金者
  - Berkshire Hathaway：负成本的浮存金+愿意在别人恐惧时承保

气候变化的真正受害者是：
  - 传统巨灾再保险公司（包括Everest）：夹在「上涨的损失」和「涌入的资本」之间
```

## 二、管理层行为信号：$376时增发而不是回购——最诚实的估值

### 核心问题

> **CEO Juan Andrade拥有最多的内部信息。他在股价$376时的行为——增发440万股、零回购——是他对Everest内在价值最真实的判断。这个判断比任何分析师报告都值得信赖。**

### 2.1 管理层行为的三层解读

| 层面 | 解读 | 置信度 |
|:---|:---|:---|
| **第一层** | Andrade认为$376高于内在价值，否则他会（也应该）回购 | ★★★★★ 几乎确定 |
| **第二层** | Andrade认为FY2025的$60 EPS不可持续，所以「便宜」的PE 6.27x是幻觉 | ★★★★☆ 高度可能 |

### 2.2 为什么Andrade要增发？

**增发的目的可能是：**

1. **融资扩张直保业务**：Andrade一直在推动Everest从纯再保险向「再保险+直保」转型，这需要资本金
2. **弥补FY2024的资产负债表损伤**：FY2024 BVPS从$130跌到$114，虽然FY2025反弹到$178，但管理层可能判断这轮反弹不稳固
3. **为潜在收购储备弹药**：再保险行业整合正在加速
4. **股权激励行权**：高管薪酬的一部分通过股票期权实现

**无论哪种原因，核心事实不变：管理层在价格$376上选择稀释股东而不是回购，这意味着CEO认为：**

> **「把股票以$376卖给新投资者，比用公司现金以$376回购注销股份，对现有股东更有利。」**

**翻译：他认为$376的价值低于公司未来能通过这些现金创造的价值。或者更简单——他认为$376不便宜。**

> **「CEO的嘴会说谎，但CEO的钱包从不说谎。零回购+增发440万股——这是Andrade用Everest的股票投下的『不信任票』。如果内部人不愿意以$376买自己的股票，为什么外部人应该买？」**

## 三、再保险百年回报率：作为资产类别，再保险公司给投资者创造了什么？

### 核心问题

> **过去100年，再保险公司作为资产类别，给长期投资者创造了什么回报？和指数基金比如何？**

### 3.1 历史数据：再保险不是一个好赛道

虽然没有精确的「全球再保险公司100年指数」，但我们可以从多个维度拼出真相：

| 投资者回报维度 | 再保险行业 | S&P 500 | 差距 |
|:---|:---|:---|:---|
| **Munich Re上市以来（~100年）年化回报** | ~7-8%（含股息） | ~10%+ | 落后2-3% |
| **Swiss Re近30年年化回报** | ~5-6% | ~10% | 落后4-5% |
| **Everest Group近20年年化回报** | ~8-9% | ~10% | 落后1-2% |
| **再保险行业平均ROE（长期周期均值）** | 8-10% | — | 仅够覆盖资本成本 |
| **最大回撤** | -50%至-70%（灾难年常见） | -50%（2008） | 但再保险的回撤更不可预测 |

### 3.2 关键结论

**再保险公司作为资产类别有两个致命缺陷：**

#### 缺陷一：长期回报低于指数

Munich Re是地球上最优秀的再保险公司——148年历史、AAA评级、全球最顶尖的精算团队。它的长期年化回报约7-8%，低于S&P 500的10%+。这意味着：**即使你选了行业内最好的公司，你的回报也不如买指数基金。**

连Munich Re和Swiss Re都跑不赢指数，Everest能吗？除非你认为Everest比Munich Re更优秀。但事实上它显然不是——规模只有Munich Re的1/6，评级低两级。

#### 缺陷二：尾部风险被系统性低估

再保险公司的回报分布极度不对称：大多数年份小幅盈利，少数年份巨额亏损。这种分布的特征是——历史平均回报看起来「还不错」，但这是因为你没有把「可能发生但还没发生」的超级灾难纳入计算。

```
Everest过去20年「看起来」年化回报8-9%。
但如果未来20年发生一次让全行业亏损$2000亿的超级灾难呢？
（2005 Katrina全行业损失约$600亿。气候变化的今天，$2000亿不是科幻。）

那么，20年的复合回报可能归零——甚至为负。
```

**这就是为什么再保险公司的PE长期低于市场均值。市场不是傻子——市场在以较低的PE对尾部风险进行折价。你以为你看到的是「便宜」，实际上你看到的是「尾部风险的对价」。**

### 3.3 再保险和指数基金的「确定性剪刀差」

| 持有20年 | S&P 500指数基金 | Everest Group |
|:---|:---|:---|
| **最悲观情景** | 年化回报0-2%（日本式失落二十年） | 年化回报-5%至-20%（超级灾难后破产/被低价收购） |
| **基准情景** | 年化回报7-9% | 年化回报2-5% |
| **最乐观情景** | 年化回报12-15% | 年化回报8-12% |
| **20年确定性** | ★★★★☆ | ★★☆☆☆ |

> **「再保险是一个100年来被反复证明『回报平庸+风险极端』的行业。这个行业的本质不是创造超额回报，而是对社会风险进行定价和转移。它的社会价值极大，但股东价值平庸。如果你想要社会价值，你可以捐款；如果你想要投资回报，你该买指数基金——或者买Berkshire Hathaway。」**

## 四、Berkshire vs Everest：根本差异不在承保，在浮存金的投资能力

### 核心问题

> **Berkshire Hathaway是全球最大的再保险承保人之一，它的保险业务创造了巨大的价值。为什么Everest不行？差异在哪里？**

### 4.1 巴菲特和李录的共同认知

李录是巴菲特和芒格的门徒。他对Berkshire的理解深度远超大多数投资者。他会一眼看穿Berkshire再保险和Everest再保险的本质差异：

> **「有人以为Berkshire的保险业务成功是因为承保做得好。错了。Berkshire的保险业务成功是因为——巴菲特可以用浮存金买苹果、可口可乐、美国运通的股票，并且永远不卖。Everest的浮存金只能买国债和投资级债券，赚2.5%。这个差异不是20%，是1000%。」**

### 4.2 五维对比：Berkshire vs Everest

| 维度 | Berkshire Hathaway | Everest Group | 差异的本质 |
|:---|:---|:---|:---|
| **承保** | 纪律严明，只在价格合理时承保。软市场时可以完全不写业务 | 必须维持市场份额和客户关系，软市场时也被迫接单 | Berkshire有「不玩的自由」，Everest没有 |
| **浮存金成本** | 长期为负（承保盈利） | 周期性正成本 | 结构性差异——不是一年，是永远 |
| **投资能力** | 巴菲特投股票、收购公司，年化回报~20%+ | 投资级债券组合，收益率~2.5% | **这是核心差距——不是10% vs 5%，是复利机器 vs 定期存款** |
| **资产负债表** | $1万亿+资产、$3000亿现金，任何灾难都是「可承受的颠簸」 | $563亿资产、$132亿净资产，一场大灾可能重伤 | 规模不是护城河——规模才是护城河的前提 |
| **管理层** | 巴菲特60年如一日，和股东利益完全绑定 | Andrade职业经理人，增发稀释股东 | 「所有者心态」vs「经理人心态」 |

### 4.3 深度分析：为什么Berkshire能做好而Everest不行？

#### 根本原因一：浮存金的「质量」完全不同

Berkshire的浮存金来自多个来源——GEICO（车险直保）、General Re（再保险）、BH Primary Group（商业保险）——而且多数是长尾业务（赔款在多年后支付）。这意味着巴菲特的浮存金不但规模巨大，而且**久期极长**。他可以安然地把浮存金投入「永远不卖」的优质企业。

Everest的浮存金主要是短尾的财产巨灾再保险。保费收进来，如果运气好一年内没有大灾，承保利润就兑现了。但这种浮存金**不能用于长期股权投资**——因为任何时候都可能需要赔付。Everest的投资组合必须保持高流动性，几乎全部是债券。

```
Berkshire浮存金久期：10-20年 → 可以投股票 → 年化回报10-20%
Everest浮存金久期：1-3年 → 只能投债券 → 年化回报2-3%

20年复利差异：
  $100 × (1.15)^20 = $1,637
  $100 × (1.025)^20 = $164

差10倍。
```

#### 根本原因二：「不玩的自由」是再保险最稀缺的能力

巴菲特做再保险最厉害的不是承保技术——是**他可以在软市场时完全退出**。1990年代软市场期间，Berkshire的再保险业务几乎归零。巴菲特宁愿闲着，也不写不赚钱的保单。

Everest不能这么做。它有固定的承保团队（工资要发）、全球客户关系（关系要维护）、以及股东对「收入增长」的预期。如果Andrade说我今年不写业务了因为费率太低——他的CEO位子撑不过下一届董事会。

**这是结构性的差异，不是管理层的差异。** 巴菲特能这么做的原因是他不需要向任何人证明自己。Everest的CEO需要向董事会和华尔街证明自己。这种差异永远无法弥合。

> **「投资再保险公司的核心矛盾——你永远不知道明年会来几场飓风。而不确定性是价值投资的敌人。为什么Berkshire能做好再保险而Everest不行？答案不在承保，在浮存金的投资能力。Berkshire把浮存金变成了苹果的股票，Everest把浮存金变成了2.5%的国债利息。一个是复利机器，一个是大额存单。」**

## 五、中美再保险市场对比

### 核心问题

> **中国的再保险市场和美国/百慕大的有什么本质不同？如果李录要在中国找一家「对标Everest」的公司，他会找到什么？**

### 5.1 结构性对比

| 维度 | 美国/百慕大市场（Everest） | 中国市场（中再集团等） |
|:---|:---|:---|
| **市场规模** | 成熟市场，巨灾再保险需求巨大（飓风、野火、地震） | 快速增长但总量仍小，自然灾害以洪水/台风为主，强度低于大西洋飓风 |
| **竞争格局** | 全球化的完全竞争市场。Berkshire、Munich Re、Swiss Re、ILS等 | 事实上的寡头垄断。中再集团占据中国再保险市场~50%+份额 |
| **定价自由度** | 国际化定价，费率周期波动剧烈 | 受监管约束较强，费率市场化程度低但稳定性高 |
| **投资自由度** | 美国债券市场深度全球第一，但再保险公司受监管限制不能大比例投股票 | 中国再保险公司可以投A股、港股，理论上投资灵活性更高，但实际投资能力存疑 |
| **气候风险** | 正面承受大西洋飓风+加州野火两大「气候最前沿」 | 中国气候风险在增长，但台风/洪水强度和频次的增长幅度低于美国 |
| **税收优势** | 百慕大零企业所得税是Everest的核心优势之一 | 中国企业所得税25%（高新企业15%），无税收套利空间 |
| **增长前景** | 成熟市场，保费增长跟随GDP | 「保障缺口」巨大——中国保险深度（保费/GDP）~4%，美国~12%，再保险增长潜力显著高于美国 |

### 5.2 中再集团（1508.HK）vs Everest Group

| 指标 | 中再集团 | Everest Group |
|:---|:---|:---|
| **市值** | ~300亿港元 | ~$280亿（~2200亿港元） |
| **P/B** | ~0.3-0.4x | 2.12x |
| **ROE** | ~5-8% | 7-19%（极不稳定） |
| **股息率** | ~4-5% | 1.86% |
| **增长** | 国内慢增长+海外扩张 | 全球承保，周期性剧烈 |
| **管理质量** | 国企，效率存疑 | 职业经理人，增发稀释 |

**李录会怎么看这个对比？**

> **「中再集团的P/B只有0.3倍，Everest的P/B是2.12倍——差了7倍。但中再集团的问题是ROE太低、管理层是国企逻辑。低估值不一定意味着便宜——有时候低估值意味着一家公司在用资本做社会福利而不是为股东赚钱。中国再保险市场的高增长不会自动转化为中再集团的高ROE——除非管理层激励发生根本性变化。」**

### 5.3 核心结论：中国没有好的上市再保险标的

如果李录要在保险赛道下注，他有两个选择：
- **在中国**：中国平安（保险 × 科技 × 生态）——不只是再保险，而是综合金融平台
- **在全球**：Berkshire Hathaway——浮存金 × 投资能力，复利机器的终极形态

他大概率不会选择纯再保险公司——不论中国还是美国。

## 六、能力圈判断：李录能「懂」再保险吗？

### 核心问题

> **再保险在李录的能力圈内还是圈外？**

### 6.1 能力圈的三层测试

李录判断一个公司是否在自己能力圈内，有三层标准：

| 测试 | 对Everest的检验 | 结果 |
|:---|:---|:---|
| **第一层：懂生意**——你知道这家公司怎么赚钱吗？ | 知道。收保费→投资浮存金→灾难时赔付。生意逻辑清晰。 | ✅ 通过 |
| **第二层：懂风险**——你知道什么情况下会亏钱，亏多少吗？ | 知道方向但不知道幅度。气候变化让灾难的频率和强度进入未知领域。精算模型的盲区就是能力圈的边界。 | ⚠️ 部分通过 |
| **第三层：能预测10年后**——你知道10年后这个生意会更好还是更差吗？ | 不知道。取决于气候变化到什么程度、ILS的竞争强度、管理层的战略选择……变量太多。不确定性是能力圈的敌人。 | ❌ 不通过 |

> **「能力圈的边界不在于你能理解什么，而在于你知道什么你无法理解。再保险的精算模型是一个黑盒子——你只能看到输入（保费）和输出（利润），但你看不到中间的假设。而恰恰是这些假设（灾难频率、相关性、气候趋势）决定了这个生意10年后的样子。如果你不能验证这些假设，你就不在能力圈内。」**

### 6.2 李录会怎么做？

李录不会因为PE 6.27x看起来很便宜就买入一家能力圈边缘的公司。他会：

1. **承认自己的局限**：再保险的尾部风险超出了他能理性评估的范围
2. **寻找替代方案**：如果想参与保险赛道，Berkshire Hathaway是更好的载体
3. **等待更好的价格**：如果硬要买Everest，至少等到灾难后P/B<1x、所有人都恐慌的时候——那时不确定性已经price in了
