# Everest Group (NYSE: EG) 风险分析——芒格逆向视角

> *"倒过来看，永远倒过来看。" ——查理·芒格*

> **核心命题：当所有人因为PE 6.27倍冲进一家再保险公司时，他们看到的是便宜货，我看到的是一个建立在沙子上的城堡——沙子是FY2025这个可能是历史上最好年份之一的利润。**

---

## 一、致命一击：Everest最可能怎么死？

再保险公司不像银行——银行死于流动性危机，再保险公司死于**模型傲慢**。

Everest Group总资产$563亿，其中$424亿是保险负债（浮存金+赔款准备金）。问题不在于这$424亿的数字本身，而在于：**这个数字是基于一个"世界会像过去一样运转"的假设计算出来的。**

如果这个假设错了呢？

下面五个场景，每一个都能独立杀死这家公司。五个一起发生？那就不只是Everest的问题了。

---

## 二、五条死亡路径

### 路径1：巨灾模型失效——"地图不是领土"

| 风险维度 | 具体内容 |
|:---|:---|
| **核心谬误** | 巨灾模型建立在历史数据上。如果气候已进入非线性变化阶段，历史数据就是最好的误导工具 |
| **典型案例** | RMS/AIR模型在2005年卡特里娜之前严重低估了墨西哥湾飓风风险；2017年哈维（"千年一遇"降雨）让所有模型措手不及；2023年夏威夷拉海纳野火——几乎没有任何模型预见到了那种火势蔓延速度 |
| **Everest暴露** | 作为全球前十大再保险公司，Everest承保从美国飓风到日本地震到欧洲洪水的全谱巨灾风险。一次"模型未预见"级别的事件足以吞噬多年承保利润 |
| **FY2025的脆弱性** | FY2025之所以是创纪录年份，**恰恰因为巨灾损失低于平均水平**。这是运气，不是技能。芒格会问：你用运气赚来的钱当作永久盈利能力来估值？ |

> **芒格锐评：** 巨灾模型是精算师给自己买安心的玩具。真正的问题是——如果你的模型假设"百年一遇"，但过去五年你经历了三次"百年一遇"，你的模型到底是错了，还是世界已经变了？如果是后者，你的整个资产负债表都需要重算。

---

### 路径2：气候变化——不是"更热"，是"更不可预测"

| 风险维度 | 具体内容 |
|:---|:---|
| **表层风险** | 更频繁、更强的飓风；更长的野火季；更极端的降水事件。这些都在发生，不再属于"尾部风险" |
| **深层风险** | **相关性爆炸**。传统再保险的核心逻辑是风险分散——佛罗里达飓风和加州野火不会同时发生。但气候变化正在打破这种负相关性。2020年就是预演：大西洋飓风季创纪录+加州史上最大野火季+全球疫情——三件事同时发生 |
| **二级效应** | 气候诉讼——州政府对化石燃料公司的诉讼正在向保险公司和再保险公司蔓延。如果法院认为承保化石燃料项目构成"气候损害知情参与"，再保险公司可能面临前所未有的法律风险 |
| **Everest的特殊脆弱性** | Everest在财产再保险领域有巨大敞口，特别是美国沿海和加州野火暴露区。问题不是"会不会有更多损失"，而是**损失分布正在从一个已知的分布变成一个未知的分布**——这意味着所有基于历史VaR/TVaR的风险管理框架都可能是纸糊的 |

> **芒格锐评：** 气候变化对再保险公司的影响不是"多一点"损失，而是让整个行业的精算基础失效。如果你的模型假设世界是静止的，而世界在加速变化，那么你的准备金就是一张彩票——只不过中奖率你完全不知道。

### 路径3：资本涌入——"好生意变成烂生意只需要太多钱"

这是最被忽视的致命风险。

| 风险维度 | 具体内容 |
|:---|:---|
| **ILS市场爆炸** | 保险连结证券（ILS）和巨灾债券市场在过去十年从不足$200亿增长到超过$1000亿 |
| **对冲基金入场** | 养老金基金、主权财富基金、对冲基金正在把再保险视为"非相关收益"资产类别。2023-2025年，这一趋势加速 |
| **费率压力** | 2023年硬市场带来的费率红利（Everest因此赚得盆满钵满）正在吸引海量新资本。经济学的铁律：**超额利润吸引竞争，竞争消灭超额利润** |
| **Everest的囚徒困境** | 如果Everest维持承保纪律（少写业务），市场份额和收入会下降；如果跟随市场降价保份额，承保利润率会崩溃。无论哪种选择，FY2025的盈利水平都不可持续 |
| **结构性劣势** | 另类资本（ILS/猫债券）的资金成本往往低于传统再保险公司的股权成本。养老金基金只需6-7%回报就满意，而Everest股东要求ROE 15%+。这种不对等竞争意味着传统再保险公司正在被系统性地挤出最优质的风险层 |

> **芒格锐评：** 再保险曾经是一个俱乐部——少数几家大公司瓜分全球风险。现在它变成了一个公开集市，任何人都可以摆摊卖风险转移。当你的竞争对手是一群只要能赚6%就开心的养老金基金时，你的15% ROE估值就是个笑话。

### 路径4：管理层信号——$376时不回购却增发

这一条是最刺眼的红旗。不需要分析，只需要看事实：

| 指标 | 数据 | 芒格解读 |
|:---|:---|:---|
| **股价** | $376.14 | — |
| **PE** | 6.27x | 表面上极便宜 |
| **P/B** | 2.12x | 不算特别便宜 |
| **回购** | **零** | 🔴 管理层不认为股票便宜 |
| **增发** | 69.9M→74.3M股（两年） | 🔴 净增发440万股，稀释6.3% |
| **BVPS一年涨56%** | $114→$178 | 🟡 大部分来自投资收益和承保利润 |
| **管理层行为** | 增发+零回购 | 🔴🔴 和PE 6.27x的表象完全矛盾 |

**芒格的核心推论：**

如果一家公司PE只有6.27倍——意味着每$6.27的股价对应$1的年度盈利——而管理层却在增发新股而不是回购，只有两种可能：

1. **管理层是傻子**（不珍惜股东价值）
2. **管理层知道一些市场不知道的事**（FY2025的盈利不可持续，甚至可能包含大量不可重复的一次性收益）

无论是哪种，作为外部股东你都不应该持有。

更具体的推论：管理层在$376增发，说明他们认为这个价格**高于内在价值**。管理层拥有最多信息，他们的行为是最高质量的信号。如果他们的信号是"卖"，你为什么在"买"？

> **芒格锐评：** 我活了99年，见过无数CEO。没有任何一个聪明的CEO会在股价"极便宜"时选择增发稀释股东而不是回购。如果你看到PE 6.27倍的公司管理层在增发而不是回购，你应该做的不是买股票，而是重新计算真实的可持续盈利。

### 路径5：周期顶峰买入陷阱——"6倍巅峰利润=30倍平均利润"

这是所有价值投资者最容易犯的错误，也是最致命的一个。

| 年份 | 净利润 | ROE | 备注 |
|:---|:---|:---|:---|
| FY2023 | $1,379M | ~10% | 正常年份 |
| FY2024 | $597M | ~4-5% | 差年份（高巨灾损失） |
| FY2025 | $2,517M | 19% | 🔥 创纪录年份 |
| 三年合计 | $4,493M | — | — |
| **三年平均** | **~$1,498M** | **~11%** | — |

**基于真实平均利润的估值：**

- PE 6.27x 基于$2,517M → 市值 = $376 × 74.3M ≈ $27.9B
- 如果用三年平均利润 $1,498M：真实PE = $27.9B / $1,498M ≈ **18.6x**
- 如果FY2026回归均值（假设$1,200M）：前瞻PE = **23.3x**

**这不再是一个"便宜"的估值了。**

芒格会说：**"6倍的巅峰利润等于30倍的平均利润。"** 意思很简单——用一个不可持续的峰值利润做估值分母，任何东西看起来都便宜。但你不是在买FY2025，你是在买FY2025之后的所有年份。

更进一步：再保险行业的利润不仅有商业周期，还有**自然灾害周期**。一个真正糟糕的飓风季可以让全行业亏掉好多年的累计利润。2005年卡特里娜+丽塔+威尔玛三连击就做到了。2017年哈维+伊尔玛+玛利亚也差不多。

FY2025是一个"天时地利"的年份——巨灾损失低、费率高位、投资收益强劲。三种好运同时发生。这三种好运**同时持续**的概率是多少？这就是你应该乘的折扣系数。

> **芒格锐评：** 人们在再保险上犯的错误和在周期性矿业股上犯的错误一模一样——在景气顶峰看到低PE就冲进去，然后在景气谷底恐慌卖出。不同之处在于，再保险的谷底可能来得比矿业更猛烈——一次飓风就可以让一年、甚至多年的利润清零。

## 三、空方论点汇总

做空再保险公司（尤其在硬市场期间）长期来看是危险的——通胀性的赔款成本和费率周期可能对你非常不利。但短期的空方论点值得理解：

| 空方论点 | 逻辑 |
|:---|:---|
| **均值回归** | 19% ROE不可持续，回归到个位数甚至负ROE时，P/B 2.12x是灾难 |
| **资本过剩** | 费率已经开始走软，2026-2027年承保利润率将大幅下降 |
| **储备不足** | 巨额利润可能掩盖了准备金计提不足——社会通胀（诉讼成本上升）正在侵蚀历史储备的充足性 |
| **气候尾部** | 一次$500亿+的巨灾事件可能让Everest一年的利润归零，而且这样的事件越来越频繁 |
| **管理层缺乏信心** | 如路径4所述，管理层自己的行为已经说明了问题 |

## 四、芒格框架：Everest的失败概率

芒格不会给你一个精确的数字，但他会用**非对称思维**来看这个问题：

```
上行空间：如果FY2025的盈利能力能持续 → PE回归到行业正常12-15x → 股价翻倍
下行空间：如果盈利回归均值+一次大灾 → ROE变负 → P/B跌破1x → 股价腰斩+

非对称比：下行风险 ≈ 上行空间（甚至更差）
```

但是，**概率呢？**

- 盈利能持续的概率：低（历史上从来没有一个硬市场是永久的）
- 会遭遇大灾的概率：中高（气候趋势+概率本身）
- 管理层在$376增发意味着内在价值<当前价格的概率：高

在芒格的框架里，当管理层的行为和财务报表的数字发生冲突时，**你信行为，不信数字。**

## 芒格的最后忠告

> *"投资不是比谁能在最危险的生意上赚最多的钱，而是比谁能在睡得好觉的情况下赚足够的钱。"*

Everest Group经营的是一门好生意吗？在硬市场期间，是的。$25亿净利润不是假的。

但是，当你在硬市场顶峰、管理层在增发而不是回购、PE低到让所有人觉得"捡到宝了"的时候买进——你就是在用永久性资本去赌一个周期性好运能持续下去。

**真正的问题不是"Everest会不会破产"（大概率不会），而是"以$376的价格买入，三年后你的年化回报是多少？"**

如果你用基于FY2025的PE 6.27x来回答这个问题，你已经在骗自己了。真实的历史平均利润显示PE应该在18-23x——这和你看到的那一个数字是截然不同的投资命题。

**芒格会等的价格：** 等到下一次硬核巨灾之后，等到所有人都认为再保险公司要完蛋的时候，等到没有人再提"PE 6.27x真便宜"的时候，等到管理层终于开始回购而不是增发的时候——那才是你需要关注的时刻。

在那之前，**看，不要碰。**
