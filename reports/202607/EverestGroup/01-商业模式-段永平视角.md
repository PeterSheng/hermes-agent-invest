## 零、段永平的保险观——他怎么看这个生意？

在分析Everest之前，必须先理清段永平对保险生意的底层认知。段永平说过："我不太懂保险，但我知道巴菲特懂。"这句话不是谦虚——段永平的投资集中在消费品（茅台、苹果）、互联网（腾讯、网易），他从未重仓过保险公司。

但这不意味着保险公司不能是好生意。段永平的框架里，好生意必须满足三个条件：

1. **产品被需要**——客户离不开你
2. **有定价权**——你能涨价，客户还买
3. **没有 leverage 风险**——你不会因为一次意外就破产

保险天然满足第一条（人人都需要保障），但在第二条和第三条上，不同保险品类的差距巨大。

**段永平会怎么看再保险？**

他会说："这生意本质上是'卖承诺'——平时你收保费很开心，但灾难来的时候你得真的赔钱。如果你算错了一笔账，可能十年赚的钱都不够赔一次。"

这也是为什么巴菲特做保险做到了全球最大，但他反复强调：保险生意只有在你**定价正确**的时候才是好生意。定价错了，保险就是用别人的钱替自己挖坟。

## 一、生意本质：再保险到底是一门什么生意？

### 一句话说清楚

> **Everest卖的是"保险公司的保险"——当保险公司承保了太多风险（飓风、地震、恐怖袭击），它们会来找Everest分担。Everest平时收保费（别人的浮存金），灾难时赔付。这生意的核心竞争力只有一个：精确的定价能力。**

### 再保险的生意链条

```
投保人(你) → 买保险 → 原保险公司(如Allstate) → 买再保险 → Everest
                                                      ↓
                                              投资浮存金赚收益
```

**Everest的"产品"是什么？** 是一纸合同——承诺在特定灾难事件发生后，向原保险公司支付约定的赔偿金。这个承诺的背后是：

1. **承保能力**（资产负债表够不够大）：总资产$563亿，净资产$132亿
2. **定价模型**（精算师能不能算对这笔风险值多少钱）：核心know-how
3. **投资能力**（收了保费之后，怎么投资这$440亿+的投资组合）：FY2025投资收益$14.3亿
4. **信誉**（出了事你赔不赔得起）：149年历史，A评级

### 这个生意的"好"与"坏"

**好的方面：**

| 优势 | 具体表现 |
|------|----------|
| **浮存金（Float）** | 先收钱、后赔付，中间可以投资赚钱。这就是巴菲特爱保险的根本原因。Everest账上有$440亿+的投资资产，投资收益年化$14亿。 |
| **必需品属性** | 保险公司必须买再保险——没有再保险，一次超级飓风就能让中小保险公司破产。监管强制要求。 |
| **门槛极高** | 新进入者几乎没有可能——需要百亿美元级的资产负债表、几十年积累的精算数据和评级、全球客户关系。 |
| **规模效应** | 越大越安全（风险更分散），越安全越能拿大单，越拿大单越大。正向循环。 |

**坏的方面：**

| 劣势 | 具体表现 |
|------|----------|
| **产品完全同质化** | Everest的再保险保单和Munich Re的没有本质区别——都是"承诺赔钱"。客户（原保险公司）选择谁，几乎只看价格和评级。 |
| **定价权极弱** | 在"软市场"（供给过剩）时，费率被疯狂压缩，所有人都在亏本抢单。Everest无法像茅台那样"每年涨一点价"。 |
| **利润极不稳定** | FY2024净利仅$5.97亿，FY2025暴增到$25.17亿——**差了4.2倍！**这不是成长，这是天气。 |
| **尾部风险** | 一次"百年一遇"的灾难（如$1500亿级别的飓风季）可能让几十年积累的利润归零——甚至让公司破产。 |
| **不可预测性** | 气候变化正在让"百年一遇"变成"十年一遇"。精算模型基于历史数据，但未来天气不在历史数据里。 |

### 段永平视角的关键判断

**这个生意的问题不在"能不能赚钱"，而在"利润能不能持续"、"风险能不能控制"。**

段永平说过一句话很适合这里：**"一个好生意是你能算出10年后大概赚多少钱的生意。"** 再保险的问题恰恰是——你算不出。你不知道明年会不会来一个卡特里娜级别的飓风，你也不知道气候变化会让飓风频率翻几倍。

巴菲特做再保险成功，是因为他有两个别人没有的东西：
1. **无限弹药**：$3000亿+的浮存金，一次超级灾难也伤不到他
2. **定价纪律**：软市场时他可以完全不接单，等硬市场来了再收割

Everest有这两样吗？**差得远。**

## 二、竞争格局：Everest在再保险行业的位置

### 全球再保险竞争格局

| 梯队 | 公司 | 市值 | 核心优势 |
|------|------|------|----------|
| **第一梯队** | Berkshire Hathaway | $万亿级 | 无限弹药+定价纪律+巴菲特本人 |
| **第一梯队** | Munich Re（慕尼黑再保险） | ~$700亿 | 全球最大专业再保，148年历史，AAA级评级 |
| **第一梯队** | Swiss Re（瑞士再保险） | ~$400亿 | 全球第二大，最顶尖的精算和风险管理 |
| **第二梯队** | Hannover Re | ~$300亿 | 德国老二，运营效率极高 |
| **第二梯队** | **Everest Group** | **~$149亿** | **百慕大注册，特殊优势：税务+灵活承保** |
| **第三梯队** | RenaissanceRe, Arch Capital等 | $50-150亿 | 细分领域专精 |

### Everest vs 巨头的差距

Everest从来不是行业第一梯队，它属于"优秀但不够卓越"的梯队。和Munich Re/Swiss Re相比：

| 维度 | Everest | Munich Re / Swiss Re |
|------|---------|---------------------|
| **规模** | 总资产$563亿 | 总资产$3000亿+ |
| **评级** | A/A+ | AA/AAA |
| **客户黏性** | 中——客户会为价格换供应商 | 高——顶级评级是硬通货 |
| **多元化** | 主要靠财产/灾难再保险 | 寿险+财险+直保全面覆盖 |
| **地域分散** | 以美国为主 | 全球均衡分布 |
| **人才/精算** | 有竞争力 | 行业最顶尖 |
| **资本纪律** | 一般（见下文稀释问题） | 较高 |

### 百慕大优势——Everest的独特生态位

Everest注册在百慕大，这不是巧合。百慕大是全球再保险"避税天堂"：

- **零企业所得税**：百慕大不对再保险公司利润征税
- **监管灵活**：比美国/欧洲更宽松的资本要求，可以更激进地承保
- **离岸资本池**：全球对冲基金、另类资本（ILS/巨灾债券）都在百慕大聚集

这意味着Everest比Munich Re有**税收成本优势**，比直保公司有**监管套利优势**。这是它能在巨头的阴影下生存的核心原因。

但这也是个脆弱的优势——如果全球税改（如OECD最低企业税率）推进，百慕大优势可能被削弱。

### 真正的竞争威胁：另类资本（ILS/巨灾债券）

过去10年，养老金、对冲基金等机构投资者大量涌入再保险市场，通过"保险链接证券（ILS）"和"巨灾债券"直接承保灾难风险。这些"华尔街资本"的特点：

- 追求6-8%的绝对回报，比传统再保险公司定价更激进
- 没有历史包袱，没有承保纪律
- 资金进出极快——软市场时跑得比谁都快，硬市场时冲得比谁都猛

**这对Everest来说是个"搅局者"**。ILS资金的进入加速了费率的周期性波动——软市场更软（大家一起亏），硬市场更短（资本快速涌入压低费率）。

## 三、定价周期：现在是"硬市场"还是"软市场"？

### 再保险的定价周期——决定一切的关键变量

再保险行业有一个铁律：**费率决定利润，费率由灾难和资本供给决定。**

```
灾难多发/资本撤离 → 费率上涨（硬市场） → 再保险公司暴赚
                                         ↓
暴赚吸引资本涌入 → 费率下跌（软市场） → 再保险公司亏损/微利
                                         ↓
亏损逼退资本 → 费率上涨（硬市场） → 循环
```

### 当前周期位置：硬市场尾部

**FY2025是Everest的利润顶峰年——ROE 19.1%，净利$25亿。** 这个数据说明了什么？

回顾历史：
- **FY2020-2022**：硬市场。疫情+自然灾害频发+资本撤离，费率连涨3年。Everest ROE从个位数爬到15%+
- **FY2023**：费率继续坚挺。ROE~15%，BVPS涨到$145
- **FY2024**：灾难年。全球巨灾损失创纪录（飓风+洪水+野火），Everest净利暴跌到$5.97亿，BVPS跌到$114
- **FY2025**：恢复年。灾难减少+前一年高费率延续+投资收益回升，净利$25亿，BVPS反弹到$178

**问题：FY2025的$25亿利润可持续吗？**

几乎肯定不可持续。原因：

1. **FY2024的巨额损失会推高FY2025的续保费率**——原保险公司在灾难后更愿意付高价买保障，这是FY2025暴赚的核心原因
2. **FY2025灾难较少**——赔付率回落，叠加高费率，利润自然爆炸
3. **但资本正在涌入**——看到$25亿利润，华尔街资本已经在摩拳擦掌。ILS基金、对冲基金正在募集更多资金进入再保险
4. **FY2026-2027费率大概率走软**——历史规律：暴赚之后必有资本涌入，资本涌入之后必有价格战

### 段永平怎么看周期？

段永平最讨厌周期。他说过：**"我喜欢那种不管经济好不好都能赚钱的生意。周期性太强的生意，你不知道什么时候该买，什么时候该卖。"**

再保险是极致周期性的生意——利润波动4倍以上。而且最大的问题是：**你不知道灾难什么时候来。**

- 如果未来3年没有大灾难，Everest每年可能赚$20-25亿
- 如果明年来了一个卡特里娜级飓风，利润可能归零甚至亏损
- 如果气候变化让灾难频率翻倍，精算模型可能彻底失效

**这种不确定性是段永平最无法接受的。** 他买苹果是因为iPhone销量可预测，买茅台是因为白酒消费可预测。再保险的利润不可预测。

## 四、管理层：Juan Andrade——合格的CEO，但不是巴菲特

### CEO背景

Juan Andrade于2019年出任Everest CEO。此前他在Chubb（全球最大上市财产险公司）担任高管20+年，负责国际业务。在Chubb期间，他以运营纪律和承保利润导向著称。

### Andrade做了什么？

自上任以来：
- **品牌重塑**：从"Everest Re"改为"Everest Group"，强调多元化
- **拓展直保业务**：通过收购进入美国个人/商业保险直保市场（不再只做再保险）
- **技术投资**：加大数据分析和精算建模投入
- **国际扩张**：进入拉丁美洲、亚洲等新兴市场

### 核心问题：资本配置质量

**⚠️ 最大红灯：股本稀释**

| 年份 | 流通股数 |
|------|----------|
| FY2023 | 69.9M |
| FY2024 | ~72M |
| FY2025 | 74.3M |

**2年时间股本增加了6.3%！** FY2025没有回购。这意味着：
- 公司通过增发股票（可能是高管股权激励、收购融资）稀释了现有股东
- 6.3%的稀释率在2年内吃掉了一大块股东回报
- 段永平极度厌恶股权稀释——苹果每年大规模回购，茅台几乎不增发

**对比巴菲特的做法：**
- Berkshire Hathaway在保险业务赚了钱之后，巴菲特把钱拿去做两件事：回购Berkshire股票（减少股本）或收购好公司
- Everest的做法是：赚了钱→扩张/收购/发股票→摊薄股东

**分红：**
- $7/股，股息率1.86%——不算差，但在$25亿利润下，分红率仅~28%
- 大量利润留在公司做再投资——问题在于，再投资的ROE是多少？

### 管理层评分：★★★☆☆

Andrade是一个合格的职业经理人，但不是"企业家型"的CEO。他做的是"把再保险公司运营得更好"，而不是"像巴菲特那样创造一种独特的竞争优势"。

段永平看重的是"right people"——这个人有没有独特的判断力、能不能在关键时刻做出反直觉但正确的决策。Andrade没有表现出这种特质。

## 五、10年确定性：10年后Everest还存在吗？会比现在更好吗？

### 10年后还存在吗？

**大概率存在。** 再保险是一个不会消失的行业——只要还有保险公司、还有灾难风险，就需要再保险。Everest有$560亿资产、A级评级、全球客户关系，不可能轻易消失。

但存在≠更好。核心问题是：**10年后Everest会比现在更强吗？**

### 三种情景分析

| 情景 | 概率 | 描述 |
|------|------|------|
| **乐观** | 25% | 气候变化推高灾难频率→硬市场持续→费率长期上涨→Everest ROE稳定在15%+→BVPS翻倍到$350+ |
| **基准** | 50% | 周期正常波动→资本涌入/退出循环→平均ROE 8-10%→BVPS缓慢增长到$250→股价随周期大幅波动 |
| **悲观** | 25% | 超级灾难+气候变化+ILS泛滥→Everest承保纪律失误→一次巨额亏损重创资产负债表→评级下调→恶性循环 |

### 10年后的关键变量

**① 气候变化——最大的"未知数"**

这是段永平最担心的问题。过去100年的精算数据建立在一个稳定的气候系统上。如果气候变化让灾难频率和强度超出历史范围，所有精算模型都会失效。

Everest承保了大量美国沿海财产再保险（飓风风险）、加州野火再保险。如果这些风险在10年内翻倍，Everest要么大幅提价（可能失去客户），要么承担巨亏。

**② 另类资本（ILS）——结构性威胁**

如果ILS市场继续膨胀，传统再保险公司会越来越像"公共事业"——赚取平庸的回报，在硬市场短暂暴利，在软市场苦苦挣扎。

**③ 管理层接班**

Juan Andrade已经做了6年CEO。如果他离开，继任者是谁？有没有可能引入一个"超级承保人"（像Ajit Jain在Berkshire那样的传奇人物）？

**④ 伯克希尔的竞争**

巴菲特一直把再保险当成"狙击枪"——只在硬市场、高价的时候才出手。如果硬市场来了，Berkshire可以瞬间承保$100亿+的保单，碾压Everest。

### 10年确定性评分：★★☆☆☆

| 维度 | 评分 | 理由 |
|------|------|------|
| 行业存在 | ★★★★★ | 再保险行业不会消失 |
| 公司存在 | ★★★★☆ | Everest大概率存活 |
| 利润可预测 | ★★☆☆☆ | 周期性强+气候不确定 |
| 比现在更好 | ★★☆☆☆ | 不确定，看气候和竞争 |

### ★★★☆☆（3星）——可理解的生意，但不可预测的利润

| 维度 | 评分 | 评语 |

| **管理层** | ★★★☆☆ | Andrade合格但不卓越，股本稀释是扣分项 |

### 段永平会说：

> **"这个生意我看得懂——卖承诺、收浮存金、投资赚钱。但我不知道明年会不会来飓风，也不知道10年后气候会变成什么样。对我来说，不会下注太多。"**

**他会买吗？**

- 如果P/B跌到1.0x以下（灾难年恐慌抛售）——**可能会小仓位试探**
- 在当前P/B 2.12x、盈利顶峰期——**不会买**
- 如果有Berkshire Hathaway作为替代选择——**为什么不买BRK？巴菲特帮你做再保险决策，而且不收管理费**

**段永平的最大问题：**

他做投资最重要的原则是：**"不懂不做"。** 再保险的"黑盒子"——精算模型、尾部风险、气候变化——对段永平来说属于"不够懂"的范畴。他知道这个生意能赚钱，但不知道这个钱能不能持续、安不安全。

相比之下，他买苹果是因为他知道iPhone每年都能卖出去——这个确定性，再保险给不了。

### 投资结论

Everest Group是一家**"好的但不是伟大的"**公司。它在正常年景能产生可观的浮存金和投资收益，在硬市场能暴赚，在灾难年会暴亏。对普通投资者来说，最大的风险不是亏钱，而是在利润顶峰时被6倍PE诱惑买入，然后发现下一年利润跌了80%。

**段永平不会重仓Everest。但如果他想在保险赛道上有一席之地，他会买Berkshire Hathaway。**
