# UMH Properties (NYSE: UMH) 商业模式深度分析：段永平视角

> **分析框架：好生意 + 对的人 + 对的价 = 长期投资的核心**
>
> 段永平常说："买股票就是买公司，买公司就是买未来现金流折现。"本报告从这一原点出发，对UMH Properties进行系统性审视。

---

## 一、一句话说清UMH的生意

**UMH = 美国预制房屋社区的"土地收租公"**——你拥有135+个社区里的土地（约25,000个地块），居民买或租预制房屋停放在你的土地上，每月向你交"地租"。你不拥有房子本身，只拥有房子下面的那块地。水电等公用事业你代收代缴，赚个差价。

> 段永平视角：这生意一句话能说清楚，符合"简单易懂"原则。比茅台还简单——茅台你还要理解品牌、渠道、基酒；UMH就是"我有地，你交租"。

### 跟茅台/苹果/可口可乐的本质区别

| 维度 | UMH | 茅台 | 苹果 | 可口可乐 |
|------|-----|------|------|----------|
| 生意本质 | 收地租 | 卖酒（品牌消费品） | 卖硬件+生态 | 卖糖水（品牌消费品） |
| 增长驱动 | 买更多地+涨租金 | 提价+放量 | 产品迭代+生态锁定 | 全球渗透+提价 |
| 复购/粘性 | 搬家成本极高 | 消费习惯/社交属性 | 生态系统锁定 | 口味/品牌忠诚 |
| 定价权 | 有限（租客收入约束） | 极强 | 极强 | 强 |
| 轻资产程度 | 重资产（需持续买地） | 轻资产 | 轻资产 | 轻资产 |
| 复利效应 | 地块收购+租金滚雪球 | 品牌复利 | 生态复利 | 品牌复利 |

**一句话总结差距：** 茅台苹果可口可乐是靠"品牌+无形资产"赚钱，每赚一块钱几乎不需要额外资本投入；UMH是靠"地+砖头"赚钱，每多赚一块钱需要先花五到十块钱买地。

## 二、需求永续性：这个生意会消失吗？

### 2.1 乐观面：结构性刚需

**美国住房负担能力危机是UMH最核心的宏观叙事。**

- 美国新建独栋住宅中位价已突破$400,000，而预制房屋的中位价约$80,000-$120,000（不含土地）
- UMH的月租金（地块租金+房屋贷款/租金）通常在$500-$900/月，而市场公寓租金中位数$1,500-$2,000+
- 美国有约2,200万人居住在预制房屋中，占总人口约6%
- 预制房屋社区是美国**最大规模的无补贴可负担住房**来源

> 段永平视角：这个需求的底层逻辑是成立的——只要美国房价收入比继续恶化，低收入群体和固定收入退休人员就需要这种住房方案。这不是一个"会不会消失"的问题，而是"有多大"的问题。类似于"中国人会不会不喝白酒了？"——会一直喝，量可能有波动。

### 2.2 隐忧面：需求的天花板

- **人口结构风险**：UMH的核心租客是蓝领工人和退休人员。制造业外迁/自动化可能侵蚀蓝领就业，影响租金支付能力
- **区域集中风险**：UMH 70%+的社区集中在"铁锈带"（俄亥俄、宾夕法尼亚、印第安纳、密歇根）——这些州人口增长缓慢甚至负增长
- **替代品竞争**：公寓开发商也在进入"可负担"细分市场，政府补贴的廉租房项目构成竞争

> **段永平判断：** 需求不会消失，但不是"永续增长"型需求。"铁锈带"的人口趋势是一个真实的风险。相比之下，茅台的需求是"中国文化+社交货币"，苹果的需求是"全球中产的身份象征"——这些需求的永续性远强于UMH。

## 三、定价权与护城河：每年能涨3-5%吗？

### 3.1 UMH的"护城河"是什么？

**（1）监管/区划护城河（Regulatory Moat）——最核心**

- 美国绝大多数城市/县**禁止新建**预制房屋社区（zoned out）
- 现有社区实际上是"永久稀缺资产"——供给几乎为零增长
- 这是UMH最接近"特许经营权"的地方

**（2）高转换成本（Switching Cost）——次要但真实**

- 移动一栋预制房屋的成本：$5,000-$15,000甚至更多
- 很多老房子根本不能移动（结构老化）
- 搬家意味着放弃房子（沉没成本$30,000-$80,000）
- **实际效果**：UMH的租客流失率极低，年化通常低于5-8%

**（3）规模/效率护城河？——不存在**

- 预制房屋社区经营不需要特别的技术或规模
- 一个社区经理+几个维修工就能管理
- 不像Costco需要全球供应链管理能力

### 3.2 定价权深度分析

**可以涨租吗？可以，但有天花板。**

- UMH历史数据显示年租金增长约3-5%，与通胀基本同步
- **但**：租客收入中位数约$30,000-$40,000/年，租金占收入比已经30-40%
- 涨租的极限是"租客还能付得起"——这不是茅台（买不起茅台的人本来就不是客户）
- 段永平心爱的提价模型：茅台提价不影响销量，苹果提价消费者骂完还买。UMH涨租$50/月，租客可能真的付不起。

> **段永平判断：UMH有护城河，但护城河的"宽度"远不如茅台苹果。区划保护是真实存在的，高转换成本也是真实存在的。但定价权的"天花板"是租客的月收入——这不是你想涨多少的问题，是别人口袋里有多少钱的问题。茅台的定价权来自"消费者愿意花更多钱"，UMH的定价权来自"消费者别无选择但也没钱"。这两种定价权，本质不同。**

## 四、商业模式深度解剖

### 4.1 轻资产还是重资产？

**是重资产，而且非常重。**

| 指标 | FY2025数据 | 解读 |
|------|-----------|------|
| 总资产 | $1.70B | - |
| 净资产 | $905.5M | BVPS $10.67 |
| 杠杆率 | ~47% | REIT中偏高 |
| 营收 | $261.75M | 完全来自资产驱动 |
| 经营CF | $82.0M | 现金流转化率约31% |
| GAAP净利 | ~$51M | 受大额折旧影响失真 |

对比茅台：总资产约￥2,700亿（$370亿），ROE约30%+，几乎零负债经营。赚来的钱不需要再投入就能继续增长。

**UMH每赚$1利润，需要先投入几块钱买地、开发、维护。** 这就是段永平最在意的"资本支出"问题。

### 4.2 地块收购的复利效应？

UMH的增长模式本质是：

```
外部融资（发股/发债/留存CF）→ 收购新社区 → 涨租金 → 产生更多CF → 再收购
```

这个飞轮**理论上**能转，但有几个问题：

- **收购价格**：好的预制房屋社区cap rate已经压缩到4-6%，以5% cap rate收购，扣除融资成本（4-5%），利差极薄
- **股权稀释**：UMH过去十年股本从约5,500万股扩张到8,485万股，稀释了约54%
- 5年营收CAGR约9%，但相当一部分来自收购+发股，内生增长（同店NOI增长）可能只有3-5%

> **段永平判断：这个"复利飞轮"转得并不漂亮。真正的复利是茅台这种——产量年增个位数、价格年增个位数、不需要额外资本，利润自己滚起来。UMH的飞轮需要不断"加油"（外部融资），而每次加油都在稀释老股东。**

### 4.3 FCF质量

REIT的FFO（Funds From Operations）更能反映真实盈利：

```
FFO ≈ GAAP净利润 + 折旧与摊销 - 资产出售收益
    ≈ $51M + ~$42M（估算折旧） - $0
    ≈ $93M
```

每股FFO ≈ $93M / 84.85M股 ≈ **$1.10/股**

P/FFO ≈ $15.52 / $1.10 ≈ **14.1x**

这个估值在REIT行业中算中等偏低。但要注意：
- FFO不包括维持性资本支出（如社区道路维修、水管更换）
- AFFO（调整后FFO）通常比FFO低10-20%
- 真实AFFO可能在$0.85-$0.95/股，P/AFFO约16-18x

> **段永平判断：一家"还行"的REIT，合理但不出彩。段永平投苹果时PE才10倍出头，但苹果的生意质量远超UMH。14倍FFO买一个ROE低、需要持续融资的生意，算不上"便宜的价钱"。**

## 五、管理层：Samuel Landy是对的人吗？

### 5.1 家族企业基因

UMH是典型的家族企业：
- **Eugene Landy**（创始人，已故）：1968年创立
- **Samuel Landy**（现任CEO，创始人儿子）：1995年起担任CEO，至今已30年
- 家族通过股票持有约5-10%的股份（具体比例需查最新Proxy Statement）
- Landy家族与公司深度绑定——**利益一致性是确定的**

### 5.2 Samuel Landy的行事风格

根据公开信息：
- **专注本业**：30年只做预制房屋社区这一件事，没有跨界瞎折腾
- **保守稳健**：不像一些REIT CEO热衷于大举杠杆豪赌
- **股东友好？**：UMH股息率约5-6%，在REIT中属正常水平。但股息增长缓慢
- **资本配置能力存疑**：过去10年股本扩张54%，稀释严重。如果收购的资产不能产生足够的增量回报，这就是在摧毁股东价值

### 5.3 对的人 vs 不错的人

> **段永平判断：Samuel Landy是"不错的人"但不是"对的人"。**
>
> 对的人（如Tim Cook、丁磊）——他们不仅诚实、专注，更重要的是有卓越的资本配置能力。Cook大规模回购苹果股票，为股东创造了巨大价值。
>
> Landy诚实、专注（30年不改行），这没问题。但他30年的成绩单：UMH股价从1995年至今年化回报率（含股息）大概在8-10%，跑赢通胀但没有惊艳表现。他没有犯大错，但也没有展现出卓越的资本配置眼光。
>
> 段永平曾说："如果管理层不够优秀，再好的生意也可能被毁掉。"反过来，UMH不是最好的生意，如果管理层也只是"还行"，那这笔投资就缺乏吸引力。

## 六、当前估值：对段永平来说贵吗？

### 6.1 估值快照

| P/B | 1.45x | REIT中合理 |

| P/FFO (估算) | ~14x | 中等偏低 |
| P/AFFO (估算) | ~17x | 中等 |
| 隐含Cap Rate | ~5.5-6.5% | 一般 |
| 股息率 | ~5.3% | REIT常态 |

### 6.2 段永平的"对的价"

段永平买公司的逻辑：
1. 先确定这是不是好生意
2. 再确定管理层对不对
3. 最后看价格是否"明显便宜"

他最经典的操作：
- 买网易：$0.8/股（后来涨到$100+），因为市场恐慌错杀
- 买苹果：~$50/股（拆股调整后），PE不到10倍
- 买茅台：~￥120/股，PE不到15倍

**核心原则：好生意+对的人，在"市场犯错"时买入。**

### 6.3 UMH现在贵吗？

> **段永平判断：不算贵，但也不便宜。更大的问题是——这不是"好生意"，所以"合理的价格"没有意义。**
>
> 段永平原话（大意）："以合理的价格买入伟大的公司，远好于以便宜的价格买入平庸的公司。"
>
> UMH现在P/FFO约14倍，对比同行Sun Communities (SUI) 约20倍、Equity LifeStyle (ELS) 约25倍，似乎"便宜"。但SUI和ELS的社区质量更高（更多分布在阳光带、增长更快、定价权更强），所以这个"便宜"可能不是真正的安全边际，而是**合理的折价**。
>
> 如果要让段永平心动，UMH可能需要跌到P/B接近1.0x或以下（即$11-12），同时确认租金增长趋势没有恶化。即便如此，他也大概率不会买——因为有更好的选择。

### 段永平最可能怎么评价？

> "这个生意我看得懂，但不想买。"
>
> 看得懂：就是出租地块，居民在上面放预制房，每月收租金。跟停车场、自助仓储、公寓一样，都是收租生意。
>
> 不想买的原因：
> 1. **客户太穷**——定价权有天花板。茅台提价不影响销量，UMH提价租客可能真的搬走。
> 2. **需要不断投钱**——赚来的利润不够买新地，还要不断发股融资。茅台赚$100，$90是自由现金流；UMH赚$100，不投回去$40-$50就维持不了竞争地位。
> 3. **没有"长长的坡，厚厚的雪"**——铁锈带人口不增长，租金增长靠通胀，球滚不大。
> 4. **管理层30年成绩单不突出**——如果Landy真的是对的人，30年应该创造更多价值。
>
> **如果非要看好预制房屋社区赛道，不如看Sun Communities (SUI) 或 Equity LifeStyle (ELS)**——它们的社区更多在阳光带（佛罗里达、亚利桑那、加州），人口增长驱动更强，租客支付能力更高，定价权更实。当然，它们也更贵（P/FFO 20-25x），但"好生意贵一点"vs"平庸生意便宜一点"，段永平永远选前者。
>
> **最终结论：不建议用真金白银投票。继续找下一个茅台、下一个苹果。**
