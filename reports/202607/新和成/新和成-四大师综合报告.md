## 五、PE/ROE快速过滤器

```
峰值PE 13.38x ÷ 峰值ROE 20.71% = 0.65  ← 芒格甜点？不，是"峰值幻觉"！
周期均值PE 21.1x ÷ 周期均值ROE 13% = 1.62  ← 这才是真正的PE/ROE

PE/ROE = 1.0 → 公平定价 → 目标价18.2元（-38%）
PE/ROE = 1.2 → 温和溢价 → 目标价21.8元（-26%）
PE/ROE = 1.6 → 当前水平 → 29.6元（偏贵但非极端）
```

> **核心洞察**：13.38x PE配20.71% ROE的PE/ROE=0.65，看起来是芒格式"甜点"。但这是用**峰值利润计算**——相当于在夏天最热的一天说"这地方气候真好"。用周期均值计算，PE/ROE=1.62，属于温和偏贵。
