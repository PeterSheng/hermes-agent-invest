## 一、产业总览：3,500亿存量博弈时代

| 维度 | 关键数据 |
|:---|:---|
| **行业规模** | ~3,500-3,547亿元（2025年） |
| **门店总量** | 突破30万家（截至2025年4月） |
| **增速变化** | 44.3%（2023）→ 12.4%（2025E，CCFA）→ 5.7%（艾媒预测） |
| **生命周期** | 从高速增长切换至 **存量博弈/洗牌期** |
| **年度开关店** | 新增~11.8万家 vs 关闭~15.7万家（净减少约4万家） |
| **行业标准** | GH/T 1495-2025《现制茶饮品原辅料要求》2026年1月实施 |
| **价格战** | 9.9元成标配，客单价降幅超60% |

> ✅ **核心判断**：行业已告别野蛮生长，价格战白热化，头部品牌通过供应链优势加速集中，中小品牌加速出清。**但竞争的核心已从"谁开店快"转向"谁供应链成本更低"。**

## 三、上游：原材料供应链 —— 「利润正在转移」

### 3.1 核心原材料分类

| 类别 | 具体品种 | 趋势 |
|:---|:---|:---|
| **茶叶** | 绿茶、乌龙、红茶、茉莉花茶基底 | 品牌自建/锁定茶园，品质标准化 |
| **水果** | 香水柠檬、芒果、葡萄、草莓、油柑等 | 区域特色水果受追捧，冷链要求高 |
| **乳制品** | 鲜奶、植脂末（奶精）、奶基底、燕麦奶 | 健康化推动鲜奶替代植脂末 |
| **糖/甜味剂** | 蔗糖、果葡糖浆、赤藓糖醇 | 代糖需求上升 |
| **小料** | 珍珠、椰果、脆波波、芋泥、奶盖 | 差异化竞争关键 |
| **包材** | 纸杯、PLA杯、封口膜、纸袋 | 环保政策趋严 |

### 3.2 主要上市/拟上市供应商

| 公司 | 主营 | 上市状态 | 关键变化 |
|:---|:---|:---|:---|
| **佳禾食品**（605300） | 植脂末（奶精） | A股主板 | 受健康化冲击，转型中 |
| **田野股份** | 原料果汁 | 北交所 | 品牌自建供应链致采购缩水 |
| **三元生物** | 赤藓糖醇 | A股 | 净利润率10%+ |
| **德馨食品** | 饮品小料 | 撤回IPO | 品牌「削藩」承压 |
| **鲜活饮品** | 果酱/糖浆 | 冲刺上市 | 蜜雪系供应商 |
| **恒鑫生活** | 纸杯/包材 | 拟上市 | 环保政策受益 |
| **南王科技** | 纸袋 | 创业板 | — |

### 🔴 核心矛盾：品牌「削藩」

> 头部品牌正在大规模自建供应链——自建茶叶基地、自主采购水果、自产核心原料，减少对第三方供应商依赖。**产业链利润正从上游供应商向中游品牌转移。** 36氪：「奶茶供应商业绩缩水，一场产业链利润的大转移」。

# 🔬 深度专题：蜜雪冰城供应链模式天花板

## 一、蜜雪冰城的本质：「披着奶茶外衣的供应链公司」

### 1.1 收入结构的真相

| 收入来源 | 占比 | 实质 |
|:---|:---|:---|
| **向加盟商销售食材/包材** | **~94%** | 供应链批发收入 |
| 设备销售 | ~3% | 一次性硬件收入 |
| 加盟费及服务费 | ~2-3% | 纯利润 |
| 直营门店奶茶销售 | **<1%** | 几乎可忽略 |

> 💡 **蜜雪冰城卖的不是奶茶，是糖蜜、果蜜、柠檬、奶精、杯子、吸管。** 每开一家加盟店，等于增加了一个「必须从蜜雪采购」的固定客户。

### 1.2 供应链自产能力

| 品类 | 自产情况 | 成本优势 |
|:---|:---|:---|
| **糖蜜/果蜜** | 全自产（大咖国际） | 包装瓶成本比外购**低49%** |
| **柠檬** | 四川安岳自建种植基地 | **掌握国内柠檬定价权** |
| **奶精/奶基底** | 核心配方自产 | 规模采购+自产压低成本 |
| **茶叶** | 锁定核心产区 | 减少中间环节 |
| **冰淇淋粉** | 全自产 | 边际成本极低 |
| **设备（制冰机等）** | 合作定制 | 标准化输出 |

### 1.3 供应链基础设施

| 环节 | 规模 | 说明 |
|:---|:---|:---|
| **中央工厂** | 大咖国际（河南）+ 多地分厂 | 核心原料全自产 |
| **仓储物流** | 全国多级仓储网络 | 覆盖所有加盟门店 |
| **海南基地** | 二期扩建中 | 服务东南亚出海 |
| **海外中心** | 规划东南亚多功能供应链中心 | 本地化供应 |

## 二、供应链模式的天花板分析

### 🟢 天花板推动力（还能涨的因素）

#### 2.1 海外扩张 → 供应链增量

| 维度 | 现状 | 空间 |
|:---|:---|:---|
| 海外门店 | ~4,700家 | 目标市场远未饱和 |
| 供应链出海 | 海南基地+东南亚中心 | 本地化生产降关税+运费 |
| 新兴市场 | 纽约首店验证 | 发达国家能否接受低价模型？ |

> 海外每开一家店 = 增加一个供应链客户。如果海外从4,700家做到2万家（假设），供应链收入将增加3倍+。

#### 2.2 品类横向扩张 → 复用供应链

| 新业务 | 现状 | 协同效应 |
|:---|:---|:---|
| **幸运咖**（咖啡） | 未成"咖啡界蜜雪" | 共用糖蜜、奶基底、包材、物流 |
| **极拉图**（冰淇淋） | 小规模 | 核心品类延伸 |
| **啤酒** | 成都青白江啤酒厂（年产3万吨） | 2025年新布局，复用渠道 |

> 💡 蜜雪的战略逻辑是：**只要是需要糖蜜、果蜜、奶精的"高频低价饮品"，都可以塞进同一个供应链管道。** 啤酒是这个逻辑的最新测试。

#### 2.3 门店密度仍有空间

- 中国县域市场仍未完全渗透
- 蜜雪官方口径：「国内门店密度低于理想水平」
- 加盟商招募持续活跃（港股IPO冻资1.77万亿港元）

---

### 🔴 天花板压制因素（被卡住的风险）

#### 3.1 单店收入稀释 —— 最致命的天花板

| 指标 | 趋势 | 含义 |
|:---|:---|:---|
| 扩店增速 | 高于营收增速 | **每家店分到的收入在下降** |
| 新店 vs 老店 | 新店分流老店客流 | 加盟商内卷加剧 |

> ⚠️ **这是蜜雪模式最核心的矛盾**：蜜雪的收入=门店数量×单店采购额。当门店密度超过临界点，**新增门店不是在开拓新市场，而是在抢老加盟商的生意**。老加盟商关店→单店采购额下降→总收入增速放缓。

#### 3.2 加盟商生存压力

| 风险 | 数据 |
|:---|:---|
| 闭店率 | ~3%（行业最低），但绝对值在上升 |
| 食品安全投诉 | 黑猫平台累计~8,727条 |
| 加盟商盈利 | 多起"37万开店不赚钱"的案例 |
| 同店竞争 | 一条街多家蜜雪的现象越来越普遍 |

> 沪上阿姨闭店率10.78%（蜜雪的3倍+），说明行业整体加盟商压力大。蜜雪虽然闭店率最低，但「6万家店的密度」正在逼近物理上限。

#### 3.3 海外复制的障碍

| 挑战 | 具体问题 |
|:---|:---|
| **本地化供应链** | 柠檬、奶源依赖国内，海外运输成本高 |
| **口味适配** | 东南亚嗜甜、欧美口味不同 |
| **品牌认知** | 低价在发达国家=低质？ |
| **文化差异** | 加盟商管理模式需本地化改造 |
| **竞争** | 东南亚本地茶饮品牌已很成熟 |

#### 3.4 第二曲线的不确定性

| 业务 | 问题 |
|:---|:---|
| **幸运咖** | 瑞幸9.9元碾压，无法复制蜜雪"柠檬水"的统治力 |
| **啤酒** | 重资产（建厂），产品形态与茶饮供应链协同有限 |

---

### 📊 天花板量化推演

```
假设变量：
  国内门店上限：8-10万家（当前~5.5万）
  海外门店上限：2-3万家（当前~4,700）
  单店年均采购额：假设维持当前水平或小幅下降
  
情景分析：
┌─────────────────────────────────────────────────────┐
│ 情景          │ 门店数   │ 单店采购额 │ 供应链收入   │
├─────────────────────────────────────────────────────┤
│ 🟢 乐观        │ 12万     │ 持平       │ 2x 当前      │
│ 🟡 基准        │ 10万     │ -10%       │ 1.5x 当前    │
│ 🔴 悲观        │ 8万      │ -20%       │ 1.0x 当前    │
└─────────────────────────────────────────────────────┘
```

> 📌 **核心结论**：蜜雪冰城的供应链模式天花板，**不取决于它能开多少店，而取决于加盟商在给定密度下还能不能赚到钱**。一旦加盟商大面积亏损关店，供应链这台印钞机就会减速。
