### 1.2 逐对手分析——谁在挖药明的墙角？

#### (A) IQVIA（昆泰+IMS）——临床CRO的800磅大猩猩 🔴🔴

| 维度 | 药明康德 | IQVIA | 胜者 |
|:-----|:--------|:------|:---:|
| 临床CRO规模 | 中等（中国第一，全球前十） | **全球第一** | IQVIA |
| 数据/分析能力 | 有限 | 全球最强（IMS健康数据） | IQVIA |
| 客户关系 | 深度绑定Top20药企 | 同样深度绑定 | 平 |
| 中国优势 | ✅ 绝对优势 | ❌ 弱势 | 药明 |
| 地缘政治风险 | 🔴 极高 | ✅ 无 | IQVIA |
| 估值 | PE 32-37x（正常化） | PE ~25x | IQVIA更便宜 |

**关键洞察：** IQVIA在临床CRO领域是药明无法超越的存在。但药明的竞争优势不在「临床」而在「一体化」——从药物发现→临床前→临床→生产，一条龙服务。问题是：**当BIOSECURE Act迫使美国药企重新思考供应链时，他们会把「一体化」的订单拆分给不同供应商吗？如果是，药明的「平台优势」就被制度性瓦解了。**

#### (B) Lonza（龙沙）——生物药CDMO之王 🔴🔴🔴

| 维度 | 药明康德+药明生物 | Lonza |
|:-----|:--------|:------|
| 生物药CDMO产能 | 快速增长 | **全球第一** |
| 技术深度 | 跟随者 | 领先者（ADC、mRNA等） |
| 客户信任 | 受BIOSECURE影响 | **零政治风险** |
| CEO稳定性 | 有代际交接问题 | 2024年CEO更换，过渡期 |
| 增长策略 | 激进扩张 | 稳健 |

**最危险的信号：** Lonza正在成为BIOSECURE Act的最大受益者。当美国药企被要求或被迫「去风险化」时，Lonza是自然的替代选择——同样的瑞士质量、零政治风险。Lonza本身不需要做什么，**药明失去的订单会自动流向Lonza。**

> **芒格会说：「竞争对手最好的日子，往往不是你变差的时候，而是外部力量帮你把对手踢出局的时候。」**

#### (C) Catalent——被Novo Holdings武装起来的睡狮 🔴🔴

Catalent在2024年被Novo Holdings（诺和诺德母公司）以165亿美元收购。这意味着：

- **资金无限**：背靠Novo Nordisk这个全球市值最高的药企之一
- **需求锁定**：Novo Nordisk的GLP-1药物（司美格鲁肽/Ozempic/Wegovy）全球爆火，需要海量CDMO产能
- **技术升级**：Novo Holdings正在向Catalent注入资本进行产能和技术升级

对药明的威胁：药明在口服固体、注射剂填充等领域的CDMO业务，将直接面对一个「资本无限+需求锁定」的竞争对手。Catalent现在不是在为利润而战，而是在为Novo Nordisk的全球供应链安全而战。

#### (D) 三星生物（Samsung Biologics）——亚洲的挑战者 🔴

三星生物在生物药CDMO领域扩张极为激进。韩国政府对生物医药产业全力支持（类似于中国当年的政策）。三星生物的竞争策略很明确：**用韩国制造替代中国制造**——在美国药企「去风险化」的叙事中，韩国的地缘政治位置比中国更安全。

#### (E) 国内对手——谁在药明的后院放火？

| 对手 | 核心业务 | 与药明的差异化 | 威胁等级 |
|:-----|:--------|:------------|:---:|
| **康龙化成** | 临床前CRO+CDMO（「小药明」） | 模式几乎一模一样，规模更小、价格更低 | 🔴🔴 |
| **泰格医药** | 临床CRO（中国第一） | 在临床CRO赛道，泰格在中国比药明更强 | 🔴🔴 |
| **凯莱英** | 小分子CDMO | 商业化CDMO产能更强，与跨国药企关系更纯粹 | 🔴 |
| **博腾股份** | CDMO | 体量较小，但在特定领域（如ADC）有特色 | 🟡 |
| **昭衍新药** | 药物安全性评价 | 细分领域龙头，药明在安评上不如昭衍 | 🟡 |

**芒格会特别关注康龙化成。** 原因：

1. 康龙化成和药明康德的商业模式几乎完全重叠——都是「临床前CRO+CDMO一体化」
2. 康龙的估值更低，增长可能更快（因为基数小）
3. 在「去风险化」叙事下，如果美国客户必须在「中国CRO」中选择一个，康龙可能因为「没有被BIOSECURE点名」（截至2026年初，康龙不在最严厉的制裁名单上）而获得意外优势

> **芒格：「如果你在一个行业里是老大，但老二和你做一模一样的事，而且价格更低——这个行业的护城河可能没有你想象的那么宽。」**

### 1.3 竞争格局的核心矛盾——「平台的诅咒」

药明康德最引以为傲的是「一体化平台」——从靶点发现到商业化生产，全链条覆盖。这个模式在2015-2022年非常成功。

**但芒格会问：这个模式在2025年以后还成立吗？**

```
            一体化平台的优势           vs           一体化平台的风险

        ✅ 客户黏性强（一站式）                     ❌ 地缘政治下「一体化」变成
        ✅ 交叉销售（CRO→CDMO）                      「系统性暴露」——整个平台
        ✅ 数据积累优势                                一颗子弹就可以打穿
        ✅ 边际成本递减                             ❌ 各环节都有更专注的对手
                                                     在蚕食
                                                   ❌ 管理复杂度指数级增长
```

**核心矛盾：药明康德的平台越成功，它在BIOSECURE Act面前的暴露面就越大。** 如果制裁只针对CDMO，药明还有CRO可以缓冲。但BIOSECURE Act针对的是整个「药明系」——CRO和CDMO一起打。平台越大，伤得越重。

## 二、失败路径 TOP 3——药明康德最可能怎么「死」？

### 🥇 失败路径一：BIOSECURE Act全面落地——美国业务断崖式崩塌

```
                    ┌─────────────────────────────────────┐
                    │         BIOSECURE死亡螺旋             │
                    │                                      │
                    │  BIOSECURE Act签署成为法律             │
                    │       ↓                              │
                    │  美国联邦机构禁止与药明签约             │
                    │       ↓                              │
                    │  大型药企开始「主动去风险」             │
                    │  （即使法律允许，合规部门也会建议退出）   │
                    │       ↓                              │
                    │  美国收入（占~60%）在3-5年内萎缩50-80% │
                    │       ↓                              │
                    │  新加坡/欧洲新产能尚未产生收入          │
                    │       ↓                              │
                    │  营收出现首次负增长                   │
                    │       ↓                              │
                    │  市场从「成长股估值」→「问题股估值」    │
                    │  正常化PE 32-37x → 12-15x            │
                    │       ↓                              │
                    │  股价腰斩至60-80元                    │
                    └─────────────────────────────────────┘
```

**概率估计：25-35%（BIOSECURE Act以严厉形式通过并产生重大影响）**

**论据：**

1. **法律进程比你想象的更近。** BIOSECURE Act在2024年已经通过了众议院委员会审议。2025-2026年，它在等待参议院版本和全体投票的时机。这种法案一旦被纳入更大的立法包裹（如国防授权法案NDAA），通过的阻力会大幅降低。

2. **不是「会不会」，而是「多严」。** 市场争论的不应该是BIOSECURE Act会不会通过，而是通过的版本有多严。温和版本可能只是限制联邦资金直接采购，给8年宽限期；严厉版本可能扩展到限制Medicare/Medicaid报销——一旦美国的医保体系不再为使用药明服务的药物买单，所有大型药企都会在24小时内终止与药明的合作。

3. **「主动去风险」已经在发生。** 即使法律尚未完全通过，大型药企的合规部门已经开始制定「去药明化」的应急预案。这不是在等法律——这是在做最坏的准备。一些药企已经在将部分项目转移到Lonza、Catalent、三星生物。

4. **收入结构极度依赖美国。** 药明康德约60-65%的收入来自美国客户。如果这个比例在5年内降至20-25%，按2025年455亿营收计算，损失约为180-200亿人民币的营收。即使中国和欧洲业务增长20-30%，也无法弥补这个缺口。

**最坏情况定量：**

| 情景 | 美国收入变化 | 总营收影响 | 正常化利润影响 | 合理PE | 目标股价 |
|:----:|:----------|:---------|:----------|:------|:------:|
| 温和（8年宽限） | -20% | -12% | -18% | 25x | 90-100元 |
| 中等（全面限制+5年） | -50% | -30% | -45% | 18x | 65-80元 |
| **严厉（Medicare/Medicaid限制+3年）** | **-75%** | **-45%** | **-65%** | **12x** | **40-55元** |

> **芒格：「如果你认为药明康德最坏只会跌20%，那你可能还没想清楚最坏情况是什么。在投资中，最坏情况通常比你想象的更坏。」**

### 🥈 失败路径二：美国模式向欧洲「复制粘贴」——地缘政治风险全球化

**概率估计：20-30%（未来5年）**

**论据：**

这是市场几乎完全没有定价的风险。目前所有人的注意力都在「中美脱钩」上，但地缘政治风险可能在向欧洲蔓延：

1. **德国/欧盟的「去风险」政策。** 欧盟在2023年提出「经济去风险」战略，虽然主要针对的是关键矿产、半导体、清洁能源，但医药供应链安全正在被越来越多地讨论。欧盟会允许自己的创新药研发高度依赖一家中国公司吗？

2. **英国的生命科学战略。** 英国脱欧后正在打造独立的生命科学生态系统。英国政府是否会跟随美国对中国生物技术公司设限？

3. **「欧洲BIOSECURE」的雏形。** 法国、德国已经在讨论「医药主权」。如果欧盟推出类似BIOSECURE的限制（哪怕程度更轻），药明的「欧洲替代计划」（在新加坡、爱尔兰、德国建厂）将受到严重打击。

4. **药明的应对策略反而暴露了脆弱性。** 药明正在新加坡和欧洲建厂，试图「去中国化」。但这恰恰证明了公司自己也认为「中国身份」是一个负担。而且——如果美国客户因为BIOSECURE离开药明，他们会去药明的新加坡工厂，还是直接去Lonza？

**如果欧洲也跟随限制：**
- 药明的海外收入占比将从~75%暴跌至~30%以下
- 新加坡和欧洲的新建产能将面临需求不足、产能利用率低下
- 公司从「全球化龙头」被迫退化为「中国本土CRO」
- 合理市值可能缩水至当前的1/3

### 🥉 失败路径三：质量/合规丑闻——数据完整性崩塌

**概率估计：15-20%（任何一年10-15%的基线风险，叠加地缘政治风险后升至15-20%）**

**论据：**

这是CRO/CDMO行业的结构性风险，与地缘政治无关——但它和地缘政治风险结合在一起，会产生乘数效应。

1. **FDA警告信的乘数效应。** CRO/CDMO行业的核心资产是「信任」——客户相信你提交的数据是真实的。一旦FDA发出警告信或发现数据完整性问题，后果不是「丢失一个客户」，而是「所有客户重新审视你」。药明历史上没有被FDA严重处罚过，但「没有发生过」不等于「不会发生」。

2. **规模越大，质量风险越高。** 药明康德员工超过4万人，在全球拥有数十个运营基地。管理和质量控制是巨大的挑战。一个人、一个环节的数据造假，就可能触发系统性的信任危机。

3. **地缘政治放大镜下。** 在没有BIOSECURE Act的世界里，药明的质量事故会被当作「一次常规的行业事故」。但在有BIOSECURE Act的世界里，同样的事故会被政治化为「中国CRO不可靠」的证据，加速客户流失。

4. **类比：华海药业（2018年缬沙坦事件）。** 华海药业是中国最优秀的原料药出口企业之一，但因为缬沙坦中发现NDMA杂质，FDA发出进口禁令，公司股价在一年内下跌超过60%。华海的出口业务至今没有完全恢复。

## 三、BIOSECURE Act 深度分析——药明的「达摩克利斯之剑」

### 3.1 法案究竟在说什么？

**核心条款（基于2024-2025年多个版本的综合）：**

| 条款 | 内容 | 杀伤力 |
|:----|:-----|:---:|
| **实体清单** | 明确列出药明康德、药明生物、华大基因等为「受关注生物技术公司」 | ⭐⭐⭐⭐ |
| **联邦采购禁令** | 禁止美国联邦机构（NIH、FDA、国防部等）与清单公司签订合同 | ⭐⭐⭐（直接影响有限，联邦采购非药明核心收入） |
| **联邦资助禁令** | 禁止接受联邦资助的机构（大学、研究型医院）使用清单公司的服务 | ⭐⭐⭐⭐（间接影响大——学术机构是新药发现的重要来源） |
| **Medicare/Medicaid挂钩** | 部分版本提议：使用清单公司服务的药物不得纳入Medicare/Medicaid报销 | ⭐⭐⭐⭐⭐（**核弹条款**——如果通过，所有大型药企必须立刻切断与药明的关系） |
| **宽限期** | 不同版本给2-8年不等的过渡期 | —（宽限期越长，药明转圜空间越大） |
| **总统豁免权** | 总统可以基于国家安全和供应链稳定进行个案豁免 | ⭐⭐（不确定性） |

### 3.2 最坏情况有多坏？

**场景：BIOSECURE Act以包含Medicare/Medicaid限制的严厉版本通过**

```
时间线：

2026年下半年  法案在国会通过，总统签署
2027年       大型药企开始将项目转移至非中国CRO/CDMO
              药明收到的新订单中美国客户占比从50%+降至20%
2028年       现有项目陆续完成或提前终止
              美国营收开始同比下降15-25%
2029年       新加坡/欧洲新产能投产，但客户拓展缓慢
              总营收首次出现两位数负增长
2030年       药明康德基本退出美国市场
              年营收从455亿降至250-300亿级别
              公司从「全球CRO龙头」变成「中国+部分欧洲CRO」
```

**收入影响估算：**

| 收入来源 | 2025占比 | 最坏情景占比 | 最坏情景营收 |
|:--------|:------:|:----------:|:----------:|
| 美国客户 | ~60% (273亿) | 10-15% | 45-68亿 |
| 中国客户 | ~20% (91亿) | 25-30% | 114-137亿 |
| 欧洲客户 | ~15% (68亿) | 12-18% | 55-82亿 |
| 其他 | ~5% (23亿) | 5-8% | 23-36亿 |
| **合计** | **100% (455亿)** | — | **237-323亿** |

**营收损失：132-218亿（-29%至-48%）**
**正常化利润（剔除一次性收益）从~100-110亿降至40-60亿（-45%至-60%）**

### 3.3 概率到底多大？

**芒格式概率判断：**

| 情景 | 概率 | 核心驱动因素 |
|:----|:---:|:------|
| 法案未能通过（国会博弈失败/总统否决） | 15-20% | 药企游说（跨国药企不希望破坏供应链）、中美关系缓和 |
| 温和版本通过（仅联邦采购限制，8年宽限期，无Medicare挂钩） | 35-45% | 这是阻力最小的政治折中方案 |
| **严厉版本通过（含Medicare/Medicaid限制，≤5年宽限期）** | **25-35%** | 国会中军工+对华鹰派的政治动能，2026年中期选举前的立法冲刺 |

> **芒格：「如果一件事有25-35%的概率让你损失40-60%的钱，而你的预期收益可能是15-20%的年化回报——这个风险回报比，值得吗？」**

## 四、空方论点——市场在定价什么？哪些合理哪些不合理？

### 4.1 空方的核心逻辑

> **「药明康德表面17.66x PE是『假便宜』——正常化PE高达32-37x，对于一个面临25-35%概率被美国市场扫地出门的公司，这个估值不是在打折，而是在为风险定价后仍然不足。BIOSECURE Act不是『会不会』的问题，而是『多严』的问题。最坏情况下，药明可能失去60%的利润。市场还没有把这个尾部风险定价充分。」**

### 4.2 空方论点逐条审视

| 空方论点 | 逻辑链 | 合理程度 | 芒格视角的评判 |
|:---|:---|:---|:---|
| **正常化PE太高** | 17.66x PE是「财务幻觉」——剔除一次性收益后真实PE 32-37x。对CRO来说不算离谱，但对「地缘政治标的」来说太贵。 | ★★★★ 高度合理 | 核心要点。如果你用32-37x PE买一家公司，你需要它未来5-10年稳定增长。药明有这个确定性吗？BIOSECURE Act说「不一定」。 |
| **BIOSECURE = 生存危机** | 法案一旦严厉通过，药明的美国业务将遭受实质性、永久性打击。这不是「跌20%抄底」的故事，而是「商业模式被法律摧毁」的风险。 | ★★★★ 高度合理 | 这是所有空方论点中最有杀伤力的一条。法律可以改变商业模式，而市场在用「法律不会通过」的假设定价。 |
| **估值没有「地缘风险折价」** | 药明的正常化PE 32-37x，高于IQVIA(~25x)、略低于Lonza(~35x)。但药明有IQVIA和Lonza没有的地缘政治风险。凭什么给同样甚至更高的估值？ | ★★★★ 高度合理 | 即使你相信BIOSECURE不会通过，合理的「不确定性折价」也应该让药明比IQVIA便宜10-20%。市场没有给这个折价。 |
| **客户正在「主动去风险」** | 大药企的合规部门已经制定了「去药明化」计划。即使法案不通过，客户的供应链多元化趋势也不会逆转。 | ★★★★ 高度合理 | 「去风险」不是法律问题，是商业常识。把所有鸡蛋放在「一个随时可能被制裁的供应商」的篮子里，任何负责任的采购总监都不会这样做。 |
| **管理层不够坦率** | 管理层在公开场合一直在淡化BIOSECURE风险，强调「客户关系稳固」、「法案不会通过」。但如果风险真的不大，为什么要在新加坡和欧洲大规模建厂？ | ★★★ 相当合理 | 言行不一。建厂行为本身证明了管理层知道风险是真实的。但对外沟通却在淡化风险——这是管理层的利益冲突：承认风险 = 加速客户流失。 |
| **国内竞争加剧** | 康龙化成、泰格医药正在追赶，且它们受BIOSECURE影响可能更小（未被明确点名）。 | ★★★ 相当合理 | 康龙化成是「影子药明」——如果美国药企必须离开药明但又不想离开中国CRO的成本优势，康龙是最自然的替代选择。 |
| **产能过剩风险** | 全球CRO/CDMO行业正在经历一轮产能扩张。如果需求增速放缓，产能过剩将压缩利润率。 | ★★☆ 部分合理 | 短期不是核心风险。药明的产能利用率仍然较高。但5年后，当新加坡/欧洲新产能+全球竞争对手新产能同时释放，可能出现过剩。 |

### 4.3 哪些空方论点不合理？

| 空方论点 | 为什么可能不对 |
|:---|:---|
| **「药明会完全退出美国」** | 即使最严厉的BIOSECURE版本，也不会一夜之间生效。美国药企对药明的依赖是真实存在的——某些复杂分子的合成能力只有药明有。完全替代需要3-5年。这不是「开关」而是「调光器」。 |
| **「中国CRO没有未来」** | 中国本土创新药研发正在爆发。即使完全失去美国市场，中国市场的增长（从低基数）也可以部分弥补。虽然不够支撑当前估值，但不至于「归零」。 |
| **「药明的技术可以被快速替代」** | 药明在CDMO领域的化学合成能力（特别是复杂小分子）是20年积累的结果。不是随便一个竞争对手就能在2-3年内复制的。工艺开发的「隐性知识」比设备更难以转移。 |

### 4.4 最有杀伤力的一条空方论点

> **「市场定价的是BIOSECURE Act『不会通过』的乐观情景。但即使法案不通过，客户已经在主动去风险。即使客户不去风险，正常化PE 32-37x在一个地缘政治风险溢出的世界里也太贵了。药明康德的三层估值假设——没有BIOSECURE、客户不会离开、高增长永续——三层都可能同时破裂。」**

## 五、认知偏误——投资药明康德最常见的思维陷阱

### 偏误①：「这次不一样」——中国CRO的「例外论」

这是药明康德投资者中最普遍的认知偏误。

```
错误推理链：

「药明康德是全球最大、最优秀的CRO/CDMO之一」
    ↓
「美国药企离不开药明康德」
    ↓
「BIOSECURE Act不会真正执行，因为不符合美国药企的利益」
    ↓
「政治会让位于商业利益」
    ↓
→ 所以药明康德是安全的

                    ↑
            这是「例外论」偏误
```

**历史告诉我们：** 政治压倒商业利益的例子比比皆是：
- 华为是全球最优秀的通信设备商——被制裁了
- 中芯国际是最先进的中国芯片制造商——被制裁了
- TikTok是最成功的中国互联网出海产品——面临强制出售

**政治逻辑 ≠ 商业逻辑。** 国会山的议员不是在帮辉瑞省钱——他们在打「中国威胁」的政治牌。药明康德是不是一个好公司，和它会不会被政治打击，是两个独立的问题。

> **芒格：「如果有人说『这次不一样』，你最好捂住你的钱包。历史上每一次重大投资灾难，都始于这四个字。」**

### 偏误②：「PE幻觉」——用假利润做真估值

这是当前最具体的认知陷阱。

```
真相：

表面PE = 3,530亿 / 192亿 = 17.66x  ← 「好便宜！」

实际PE = 3,530亿 / (192亿 - 一次性收益)
       ≈ 3,530亿 / ~95-110亿
       ≈ 32-37x                  ← 「不便宜了……」
```

**投资者的心理过程：**
1. 打开股票软件，看到PE(TTM) = 17.66x
2. 对比一下迈瑞医疗(~21x)、爱尔眼科(~40x)
3. 得出结论：「药明康德好便宜！CRO龙头才17x！」
4. 买入

但17.66x PE是基于192亿的利润——这个利润包含了大量的非经常性收益（资产处置、投资收益、公允价值变动等）。剔除这些后，公司真正的「经营利润」可能在95-110亿之间。

**芒格会说：** 「用非经常性收益做分母来估值，相当于用中彩票的钱来计算你的年收入。你可以中一次彩票，但你不能靠中彩票过一辈子。」

### 偏误③：「锚定效应」——172元的高点成为心理锚点

```
2021年股价高点：  ~172元   ← 锚点
当前股价：        ~118元   ← 「从高点跌了31%！一定被低估了！」
未来可能：        ~60-80元 ← 「不可能！比高点跌了65%？」
```

这是典型的行为金融学陷阱。投资者天然地把历史高点当作「价值锚点」，从高点跌幅越大，越觉得「便宜」。但公司的价值变化和股价的历史高点无关。

**关键问题：2021年的172元对应的是什么？**
- 2021年全球生物医药牛市（COVID推动）
- 零地缘政治风险
- 高增长预期（25-30% CAGR）

**而2026年的药明：**
- 增长放缓（15-20%）
- BIOSECURE Act悬在头顶
- 全球生物医药融资环境收紧

**用2021年的172元作为「价值锚」来衡量2026年118元是否便宜，就像用2021年的房价来判断2026年的房价——基础条件全变了。**

### 偏误④：「低负债=安全」——混淆了「不会破产」和「不会让股东亏钱」

```
药明康德资产负债率：22%  ← 「好安全！不会破产！」

但是：
- 会破产 ≠ 会让股东亏钱
- 不会破产 ≠ 股价不会跌50%
```

**资产负债率低意味着公司偿付能力强，不会因为债主逼债而倒闭。但它不能保护股东免受以下损失：**
- 收入因为地缘政治而萎缩
- 利润因为客户流失而下降
- 估值因为成长叙事破裂而收缩

**药明康德不需要破产来让投资者亏钱。** 它只需要：
1. BIOSECURE Act通过（概率25-35%）
2. 美国收入萎缩50%
3. 正常化利润从~100亿降至~40亿
4. PE从32-37x杀至12-15x
5. 股价跌至40-60元

这些都不需要「破产」——只需要「不再是一家全球公司」。

### 偏误⑤：「客户黏性=无法替代」——混淆了「不愿走」和「不能走」

```
错误逻辑：
「全球前20大药企几乎都是药明的客户」
    ↓
「他们已经深度绑定了」
    ↓
「他们不会因为一纸法案就离开」
    ↓
→ 所以药明是安全的

        ↑
    混淆了「不愿」和「不能」
```

**真相：** 药明的客户确实「不愿」离开——切换供应商需要成本、时间、质量风险。但如果BIOSECURE Act要求他们离开（特别是如果Medicare/Medicaid报销与供应商挂钩），他们不是「不愿走」而是「不能留」。

**区分：**
- **没有法律强制时：** 客户黏性=不愿离开 → 高转换成本 → 护城河 ✓
- **有法律强制时：** 客户黏性=不能留下 → 高转换成本成为过去 → 护城河被法律摧毁 ✗

> **芒格：「客户黏性是最好的护城河——直到政府把它炸掉。」**

### 偏误⑥：「好的管理层=好的投资」——管理层的利益和股东不完全一致

药明的管理层（李革、赵宁家族等）毫无疑问是卓越的企业家——从一间实验室做到全球最大CRO之一。但是：

| 管理层行为 | 股东应该问的问题 |
|:--------|:---------|
| 持续减持 | 如果公司前景那么好，为什么创始人在卖？ |
| 多元化投资（药明生物、药明巨诺等） | 管理层的精力是分散的吗？每个子公司都值得投资吗？ |
| 对外谈化BIOSECURE风险 | 是真的不担心，还是承认风险会加速客户流失？ |
| 新加坡/欧洲建厂 | 这些投资的ROIC能达到中国业务的历史水平吗？ |

**芒格的提醒：** 「好管理层≠好投资。你需要好管理层+好生意+好价格。药明有一个好管理层和一个好生意，但价格可能不好——因为价格里包含了『地缘政治风险不会爆发』的乐观假设。」

## 六、芒格视角总结：药明康德的「脆弱性公式」

```
                    地缘政治暴露面 × 政策不确定性
脆弱性 ∝ ─────────────────────────────────
                  技术不可替代性 × 客户多元化


其中：
• 地缘政治暴露面 ↑↑↑ → 美国收入占60%，BIOSECURE直接标的 → 脆弱性极高
• 政策不确定性 ↑↑ → 法案在立法进程中，结果未知 → 不确定性最大
• 技术不可替代性 ↑ → 部分复杂分子合成只有药明能做 → 脆弱性有所降低
• 客户多元化 ↓ → 过于依赖美国 → 脆弱性上升
• 资产负债率22% → 财务安全性好 → 但在法律风险面前，财务安全不等于投资安全
```

### 多空思考

**如果你是多头，你需要相信：**
1. BIOSECURE Act不会通过，或者会以极弱形式通过（概率：看多的人估计60-70%，我估计15-20%）
2. 即使通过，客户因为技术依赖不会大规模离开（与历史证据冲突——合规部门的决定优先于采购部门）
3. 中国和欧洲业务的增长能弥补美国的任何损失（定量分析不支持——缺口太大）
4. 市场会在法案尘埃落定后重新给「正常化PE 25-30x」的估值（取决于法案结果）

**如果你是空头，你只需要相信：**
1. 25-35%的概率BIOSECURE严厉通过 → 利润可能腰斩 → 当前估值太贵

> **多头需要5条都成立，空头只需要1条成立。**

### 芒格可能会给药明康德投资者的最终建议

> **「药明康德是一个好生意——在错误的地点。如果这家公司总部在瑞士，我应该会愿意为它付35倍PE。但它在中国，而且是被美国国会点名的那一个。这就像在一条美丽的沙滩上建了一座最好的房子——唯一的问题，是沙滩上写着『海啸疏散路线』。」**

> **「32-37倍正常化PE买入一家面临25-35%概率损失一半利润的公司——这在数学上不成立。即使你赢了（BIOSECURE不通过），你赚的是15-20%的年化回报。但如果你输了，你亏的是50-60%。这不叫投资，这叫『用不对称的下行风险换取平庸的上行回报』。」**

> **「我不会做空药明康德——做空一个好生意永远不是一个好主意。但我也不会买入——直到BIOSECURE Act有一个明确的结局，或者股价跌到足以补偿尾部风险的便宜程度。在投资中，『不知道』和『知道是坏的』一样危险。药明康德的未来取决于一纸法案——而我不会假装我知道美国国会下一票投给谁。」**
