# Hermes Agent Invest — 四大师价值投资研究系统

基于 Hermes Agent 的巴菲特/芒格/段永平/李录四维价值投资分析框架，自动化投研报告生成。

> **分析时段：2026年4月-7月 | 824份报告 | 195个公司/主题目录**
> 
> GitHub: https://github.com/PeterSheng/hermes-agent-invest

---

## 框架

| 大师 | 核心关注 | 视角 |
|------|---------|------|
| 段永平 | 好生意 + 对的人 + 对的价 | 商业模式本质、用户价值、定价权 |
| 巴菲特 | 护城河 + 安全边际 | 护城河五类验证、财务质量审计、内在价值 |
| 芒格 | 逆向思考 | 失败路径、认知偏误、反共识 |
| 李录 | 文明级趋势 | 范式转移、10-20年确定性、管理层文化 |

## 工具

- `tools/financial_rigor.py` — 财务数据精确验算（市值、PE/PB/ROE、三情景估值、DCF、Benford检测）
- `tools/ashare_data.py` — A股行情与财务数据（实时行情 + 近5年财务 + 估值指标）
